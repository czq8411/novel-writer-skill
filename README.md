## ✨ 核心特性

- **6大模式按需切换** — 灵感构思师 / 大纲架构师 / 细纲工程师 / 正文执笔者 / 包装运营师 / 数据复盘师，输入触发词即可切换
- **三大核心铁律** — 去AI味（禁忌词黑名单+质感描写词库）、沉浸式体验（五感调用+近距离视角）、数据导向（完读率/追读率/爽点密度）
- **MCP工具联动** — Memory（记忆管理）、Sequential Thinking（逻辑推演）、Filesystem（文件管理）
- **角色灵魂注入** — 矛盾塑造、侧面揭示、选择揭示本质、缺陷致命化、反派深度塑造
- **场景描写技法** — 镜头语言（远景→特写）、环境情绪映射、动态/静态场景、感官调用模板
- **对话写作技法** — 潜台词设计、角色语音辨识度、对话节奏控制、对话与动作黄金比例
- **AB测试+质量校验闭环** — 关键场景多版本生成，偏好学习+自动质量校验联动
- **章节节奏自动评分** — 句式/段落/剧情推进/情绪曲线4维量化评分+优化方案
- **跨卷连续性检测** — 卷间衔接、人物状态追踪、伏笔跨卷追踪、力量体系一致性
- **创作数据分析仪表板** — 进度概览、爽点密度曲线、人物出场频率、质量趋势
- **角色关系动态图谱** — 6种关系类型、动态追踪表、转折点设计、可视化关系网络
- **创作灵感日推** — 基于偏好的智能推荐，每日推送定制化创作灵感
- **渐进式新手引导** — L1-L4用户等级识别，新手6步路线图，术语解释卡
- **用户偏好系统** — 跨会话记忆，自动学习创作偏好
- **10种写作风格库** — 克苏鲁/赛博朋克/史诗奇幻/热血武斗/都市言情/悬疑推理/历史正剧/科幻硬核/末世生存/轻松日常
- **8种类型小说专项模板** — 玄幻升级流/都市异能/言情/悬疑推理/末世生存/系统流/历史架空/科幻硬核

### 模式详解

| 模式 | 功能 | 核心能力 |
|------|------|---------|
| **模式A：灵感构思师** | 从零生成创意方案 | 市场定位、差异化设计、瓶颈突破工具箱、灵感日推 |
| **模式B：大纲架构师** | 构建7模块核心大纲 | 世界观、力量体系、人物设定、情节线、角色关系动态图谱 |
| **模式B-2：细纲工程师** | 章节级拆解规划 | 伏笔管理、大纲一致性校验、爽点指令 |
| **模式C：正文执笔者** | 创作章节正文 | 场景描写技法、角色灵魂注入、对话技法、AB测试闭环、自动校验 |
| **模式D：包装运营师** | 书名/简介/标签 | 点击率优化、简介转化率、黄金三段式 |
| **模式E：数据复盘师** | 全方位诊断 | 数据分析仪表板、7维诊断、跨卷检测、节奏评分、读者视角模拟 |

## 🎯 设计理念

### 三大核心铁律

1. **去AI味与拟人化** — 禁忌词汇黑名单、动词为王、感官落地、句式破碎、近距离视角、潜台词对话
2. **沉浸式体验** — 开篇即入戏、动态叙事、视角约束、五感调用
3. **数据导向** — 完读率/追读率/点击率/转化率，黄金三章，钩子机制，爽点密度

### 架构设计

- **宪法架构**：核心铁律作为唯一真相源，各模式引用而非重复
- **懒加载**：按触发词加载对应模式子文件，避免全量加载导致注意力分散
- **模式切换保护**：检测前置条件，防止跳过必要步骤
