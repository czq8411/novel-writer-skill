## 🎯 设计理念

### 三大核心铁律

1. **去AI味与拟人化** — 禁忌词汇黑名单、动词为王、感官落地、句式破碎、近距离视角、潜台词对话
2. **沉浸式体验** — 开篇即入戏、动态叙事、视角约束、五感调用
3. **数据导向** — 完读率/追读率/点击率/转化率，黄金三章，钩子机制，爽点密度

### 架构设计

- **宪法架构**：核心铁律作为唯一真相源（SKILL.md第二章），各模式引用而非重复
- **三级懒加载**：SKILL.md（~170行）→ 模式核心文件（~150-300行）→ 子模块文件（~100-300行），按需逐级加载
- **模式切换保护**：检测前置条件，防止跳过必要步骤
- **注意力优化**：主控文件从703行压缩至171行（-76%），模式C从832行压缩至232行（-72%），模式E从1037行压缩至153行（-85%）
