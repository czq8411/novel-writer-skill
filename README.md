## ✨ 核心特性

- **7大模式按需切换** — 灵感构思师 / 大纲架构师 / 细纲工程师 / 正文执笔者 / 包装运营师 / 数据复盘师 / 全自动流水线
- **三大核心铁律** — 去AI味、沉浸式体验、数据导向，SKILL.md为唯一真相源
- **MCP工具联动** — Memory（记忆管理）、Sequential Thinking（逻辑推演）、Filesystem（文件管理）
- **模块化架构** — 主控文件~170行，shared/目录存放通用模块，modes/子目录存放模式子模块，按需加载
- **角色灵魂注入** — 矛盾塑造、侧面揭示、缺陷致命化、反派深度塑造
- **对话写作深度技法** — 潜台词设计四法、快慢节奏对话、沉默戏剧力量、权力博弈、对话动作协同
- **场景描写实战技法** — 镜头语言、环境情绪映射、多题材"稀薄→充实"对比示例、扩充三大技法
- **章节开头技巧体系** — 十种强力开头技巧+三题材网文示例
- **悬念钩子分类体系** — 十三种钩子类型+网文示例+钩子强度检查
- **内容扩充技巧** — 字数不足诊断、防注水原则、五大扩充技法
- **情节结构模板库** — 三幕式、英雄之旅、类型专用结构、网文特色结构、反转/多线叙事
- **题材差异化指南** — 玄幻/言情/悬疑/科幻/历史五大题材专项指导
- **创作模板市场** — 退婚流/废材逆袭/重生复仇/系统流/穿越种田五大模板
- **并行写作模式** — 长篇30章+支持子Agent分批并行创作
- **快捷入口** — 信息丰富时跳过问答，直接开始规划
- **AB测试+质量校验闭环** — 关键场景多版本生成，偏好学习+自动质量校验联动
- **章节节奏自动评分** — 句式/段落/剧情推进/情绪曲线4维量化评分+优化方案
- **跨卷连续性检测** — 卷间衔接、人物状态追踪、伏笔跨卷追踪、力量体系一致性
- **创作数据分析仪表板** — 进度概览、爽点密度曲线、人物出场频率、质量趋势
- **角色关系动态图谱** — 6种关系类型、动态追踪表、转折点设计
- **创作灵感日推** — 基于偏好的智能推荐，每日推送定制化创作灵感
- **渐进式新手引导** — L1-L4用户等级识别，新手6步路线图
- **用户偏好系统** — 跨会话记忆，自动学习创作偏好，中断续写智能检测
- **创作里程碑系统** — 字数/章节/连续更新/质量/剧情5类里程碑
- **章节自动编号与项目管理** — 自动创建目录结构、章节编号、细纲目录维护
- **智能章节过渡检测** — 场景承接/时间连贯/情绪连贯/钩子回应4维检测+自动修复
- **创作过程可视化仪表板** — 进度条/状态看板/每日写作日历/健康度仪表/风险预警
- **读者评论模拟** — 爽文读者/逻辑党/角色粉3类读者视角模拟
- **多平台简介适配** — 起点/番茄/晋江/纵横4平台简介风格差异化
- **书名A/B测试** — 多方案点击率预估+平台适配推荐
- **标签SEO优化** — 核心/题材/辅助/长尾4层标签体系
- **细纲质量自动评分** — 5维评分，低于阈值自动重写
- **细纲反向验证** — 从细纲反推大纲，检测偏离并自动修复
- **多章联动优化** — 检测相邻章节的节奏/情绪/伏笔/视角/信息密度联动

- **章节版本管理** — 自动保存历史版本，支持版本对比、回退和清理
- **创作灵感碰撞工作坊** — What-If接力/角色即兴对话/世界观压力测试/跨题材融合实验
- **多模态创作可视化** — ASCII风格关系图/世界地图/时间线/爽点热力图/情绪曲线叠加
- **AI写作伦理与原创性保障** — 原创性检测、AI参与度透明化、创作者权益保护

### 模式详解

| 模式 | 功能 | 核心能力 |
|------|------|---------|
| **模式A：灵感构思师** | 从零生成创意方案 | 市场定位、差异化设计、瓶颈突破工具箱、灵感碰撞工作坊、灵感日推 |
| **模式B：大纲架构师** | 构建7模块核心大纲 | 世界观、力量体系、人物设定、情节线、角色关系动态图谱 |
| **模式B-2：细纲工程师** | 章节级拆解规划 | 伏笔管理、大纲一致性校验、质量自动评分、反向验证、多章联动优化、模板库 |
| **模式C：正文执笔者** | 创作章节正文 | 场景描写技法、角色灵魂注入、对话技法、AB测试闭环、自动校验、跨章节校验修复闭环、章节版本管理 |
| **模式D：包装运营师** | 书名/简介/标签 | 多平台适配、书名A/B测试、标签SEO、封面文案、简介数据驱动优化 |
| **模式E：数据复盘师** | 全方位诊断 | 数据分析仪表板、8维诊断、跨卷检测、节奏评分、可视化看板、读者评论模拟、多模态可视化 |
| **模式F：全自动流水线** | 一键全流程创作 | A→B→B-2→C→E→D自动串联，关键决策点暂停确认 |

## 🎯 设计理念

### 三大核心铁律

1. **去AI味与拟人化** — 禁忌词汇黑名单、动词为王、感官落地、句式破碎、近距离视角、潜台词对话
2. **沉浸式体验** — 开篇即入戏、动态叙事、视角约束、五感调用
3. **数据导向** — 完读率/追读率/点击率/转化率，黄金三章，钩子机制，爽点密度

### 架构设计

- **宪法架构**：核心铁律作为唯一真相源（SKILL.md第二章），各模式引用而非重复
- **三级懒加载**：SKILL.md（~170行）→ 模式核心文件（~150-300行）→ 子模块文件（~100-300行），按需逐级加载
- **模式切换保护**：检测前置条件，防止跳过必要步骤
- **注意力优化**：主控文件从703行压缩至171行（-76%），模式C从832行压缩至232行（-72%），模式E从1037行压缩至153行（-85%）
