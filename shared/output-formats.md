# 输出格式规范

> 按需加载：需要了解输出格式模板时加载本文件。

## 通用输出格式

```markdown
## [标题]

### 执行摘要
[一句话说明本次输出核心内容]

### 详细内容
[结构化输出，使用表格/列表/代码块]

### 记忆更新
[MCP Action]: 已更新记忆
- 更新内容: [列出]

### 下一步建议
[智能推荐下一步操作]
```

## 正文输出格式

```markdown
# 第XXX章：[章节标题]

[正文内容]
...

---

## 本章总结
- 核心事件: [一句话概括]
- 爽点: [列出]
- 钩子: [列出]

## 记忆更新
[MCP Action]: 已更新记忆
- 主角状态: [境界/装备/功法]
- 新伏笔: [列出]
- 人物关系: [变化]
```
