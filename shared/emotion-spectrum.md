## 一、情感光谱体系

### 1.1 情感强度光谱（7级）

| 级别 | 名称 | 特征 | 语言特征 |
|:---:|------|------|---------|
| 1 | 微露 | 细微动作暗示 | 中性词汇、简短句式 |
| 2 | 浅显 | 表情或小动作表达 | 轻度情感词汇、中短句 |
| 3 | 明显 | 他人可直接感知 | 明确情感词汇、感叹句 |
| 4 | 强烈 | 情感占据主导 | 强烈情感词汇、反问句 |
| 5 | 爆发 | 情感失控 | 感叹句、排比句、省略号 |
| 6 | 崩溃 | 失去理性 | 破碎句式、重复词语 |
| 7 | 死寂 | 情感耗尽后的空洞 | 极简语言、大量留白 |

**强度升级原则**：避免从1级直接跳到6级（极端突发事件除外）；不同角色情感阈值不同。

### 1.2 核心情感类型（12种）

| 情感类型 | 核心感受 | 触发场景 | 描写要点 |
|---------|---------|---------|---------|
| 喜悦 | 愉悦、满足、兴奋 | 达成目标、获得奖励、重逢 | 轻快动作、明亮环境 |
| 悲伤 | 难过、失落、心碎 | 失去重要的人/物、失败 | 缓慢动作、灰暗环境 |
| 愤怒 | 生气、愤慨、暴怒 | 被冒犯、被背叛、不公 | 短促动作、尖锐语言 |
| 恐惧 | 害怕、不安、恐慌 | 面对危险、未知、威胁 | 急促呼吸、紧绷肌肉 |
| 惊讶 | 意外、震惊、错愕 | 突发状况、反转、秘密揭露 | 停顿、疑问句、感叹句 |
| 厌恶 | 反感、嫌弃、憎恶 | 肮脏、虚伪、违背价值观 | 负面形容词、抗拒动作 |
| 羞愧 | 内疚、自责、尴尬 | 犯错、被揭穿、失败 | 结巴、小动作增多 |
| 嫉妒 | 羡慕、不甘、怨恨 | 他人获得自己想要的东西 | 酸溜溜的语言、对比描写 |
| 焦虑 | 担忧、紧张、不安 | 不确定的未来、压力 | 碎片化思绪、重复动作 |
| 平静 | 安宁、满足、释然 | 达成和解、内心平和 | 舒缓语言、平和环境 |
| 期待 | 盼望、憧憬、紧张等待 | 即将发生重要事情 | 充满希望的语言、倒计时描写 |
| 迷茫 | 困惑、不知所措 | 失去方向、重大抉择 | 疑问句、矛盾动作 |

### 1.3 复合情感组合

| 复合情感 | 组成 | 描写要点 |
|---------|------|---------|
| 悲喜交加 | 悲伤+喜悦 | 流泪的微笑、矛盾动作 |
| 又爱又恨 | 爱意+愤怒 | 温柔动作中带着僵硬 |
| 哭笑不得 | 无奈+好笑 | 嘴角抽搐、想笑又叹气 |
| 爱恨交织 | 深爱+憎恨 | 眼神复杂、欲言又止 |
| 百感交集 | 多种情感混杂 | 沉默、表情变化复杂 |

## 二、情感描写技法

### 2.1 情感表达三要素

- **动作暗示**（首选）：通过人物动作间接表达情感  
  *例：她攥紧了衣角——这是她紧张时的习惯。*
  
- **环境映射**：用环境描写烘托人物情感  
  *例：窗外的雨越下越大，敲打着玻璃，像是她此刻纷乱的心。*
  
- **感官描写**：通过感官体验传递情感  
  *例：闻到他身上熟悉的烟草味，她的眼眶突然就热了——那是父亲的味道。*

### 2.2 情感节奏控制

| 节奏类型 | 情感强度变化 | 适用场景 |
|---------|-------------|---------|
| 渐强 | 从弱到强逐步升级 | 情绪积累爆发 |
| 渐弱 | 从强到弱逐步平复 | 情绪宣泄后 |
| 突转 | 瞬间从一种情感转为另一种 | 反转场景 |
| 交织 | 多种情感同时存在 | 复杂心理 |

### 2.3 情感描写禁忌

| 禁忌 | 错误示例 | 正确示例 |
|------|---------|---------|
| 直接陈述情感 | 他很悲伤。 | 他望着窗外，一动不动，只有指尖在微微颤抖。 |
| 滥用情感词汇 | 她非常非常开心，开心得跳了起来。 | 她的眼睛亮了起来，嘴角抑制不住地向上扬。 |
| 情感突兀 | 上一章还在笑，这一章突然痛哭。 | 需要过渡：笑容渐渐凝固，眼眶慢慢红了... |
| 情感过度 | 一点小事就哭得死去活来。 | 情感强度应与事件严重程度匹配 |

## 三、催泪与燃点场景设计

### 3.1 催泪场景模板

**生离死别**：
```
[人物A]的手渐渐松开，眼神失去了光彩。
[人物B]紧紧抱着他，一遍遍地喊着他的名字，声音越来越嘶哑。
周围的一切都变得模糊，只有那声微弱的"保重"还在耳边回响。
```

**牺牲与奉献**：
```
他推开她，独自冲向了危险："你快走！这里交给我！"
她想拉住他，却只抓住了一片衣角。
看着他的身影消失在火光中，她终于忍不住失声痛哭。
```

### 3.2 燃点场景模板

**逆袭时刻**：
```
所有人都以为他会倒下的时候，他缓缓站了起来。
"你们以为这样就能打败我？"他的声音不大，却带着一股不容置疑的力量。
光芒从他体内爆发，整个天地都为之变色。
```

## 四、情感诊断与优化

### 4.1 情感密度检测

| 检测项 | 标准 | 优化建议 |
|--------|------|---------|
| 单章情感峰值 | 每章至少1个 | 增加情感爆发点 |
| 情感多样性 | 避免全篇只有一种情感 | 增加情感层次 |
| 情感真实性 | 情感与场景匹配 | 调整情感强度 |
| 情感过渡 | 情感变化有合理过渡 | 增加过渡描写 |

### 4.2 优化建议

| 问题类型 | 表现 | 优化方案 |
|---------|------|---------|
| 情感单薄 | 描写空洞，没有细节 | 增加动作、环境、感官描写 |
| 情感失真 | 情感与场景不匹配 | 调整情感强度或场景描写 |
| 情感突兀 | 情感变化缺乏过渡 | 增加过渡段落或伏笔 |
| 情感泛滥 | 情感过于密集 | 适当留白，给读者喘息空间 |
