## 一、通用结构模板

### 1.1 三幕式结构

网文长篇最常用的底层结构，适用于50章以上的长篇创作。

| 幕 | 章节占比 | 核心功能 | 关键节点 |
|----|---------|---------|---------|
| **第一幕：设置** | 25% | 建立世界观、引入主角、触发事件 | 日常世界→金手指觉醒/穿越→第一次冲突→不可逆决定 |
| **第二幕：对抗** | 50% | 升级成长、势力碰撞、伏笔展开 | 初次失败→获得导师/资源→实力提升→中期大高潮→最大失败 |
| **第三幕：结局** | 25% | 最终对决、伏笔回收、主题升华 | 绝境重生→最终决战→代价与收获→新世界秩序 |

**网文适配调整**：
- 第一幕压缩至15-20%（网文读者耐心有限，需更快进入主线）
- 第二幕拆分为3-5个"小幕"，每个小幕有独立的小高潮
- 第三幕保留完整，但增加"新世界展开"作为续集钩子

### 1.2 英雄之旅（12阶段）

适用于主角成长线清晰的作品，尤其适合废材逆袭、重生复仇类。

| 阶段 | 说明 | 网文对应 |
|------|------|---------|
| 1. 平凡世界 | 主角的日常状态 | 废材/赘婿/普通人的初始身份 |
| 2. 冒险召唤 | 改变命运的契机 | 金手指觉醒/穿越/重生/系统降临 |
| 3. 拒绝召唤 | 主角的犹豫或恐惧 | 怀疑金手指/害怕改变/想回到过去 |
| 4. 遇见导师 | 获得指引或资源 | 老爷爷/系统引导/前世记忆 |
| 5. 跨越门槛 | 正式进入新世界 | 第一次使用金手指/离开新手村 |
| 6. 考验与敌人 | 初步挑战和盟友 | 第一次打脸/收小弟/建立势力 |
| 7. 接近核心 | 接近最大挑战 | 发现更大世界/接触核心秘密 |
| 8. 最大磨难 | 最大失败或损失 | 被更强对手碾压/失去重要之人 |
| 9. 获得奖赏 | 从失败中获得关键突破 | 突破瓶颈/获得终极传承 |
| 10. 返回之路 | 带着力量回归 | 回归复仇/清算旧账 |
| 11. 最终复活 | 最终考验和重生 | 最终决战/置之死地而后生 |
| 12. 带着宝物归来 | 完成转变，获得自由 | 成为最强者/建立新秩序 |

## 二、类型专用结构

### 2.1 悬疑结构

| 阶段 | 章节占比 | 核心任务 | 读者心理 |
|------|---------|---------|---------|
| **谜题出现** | 1-5章 | 引入核心谜团，建立 stakes | "发生了什么？" |
| **调查深入** | 6-20章 | 收集线索，排除错误方向，引入红鲱鱼 | "我猜是A……不对，是B？" |
| **转折点** | 21-25章 | 关键线索出现，颠覆之前所有推测 | "原来如此！但等等……" |
| **真相揭示** | 26-30章 | 揭示真相，回收所有伏笔 | "竟然是这样！" |

**悬疑写作要点**：
- 线索必须公平呈现（读者理论上能推理出真相）
- 红鲱鱼（误导线索）必须有合理解释，不能是作者耍赖
- 真相必须比读者的推测更精彩，否则读者会失望

### 2.2 言情结构

| 阶段 | 章节占比 | 核心任务 | 情感节奏 |
|------|---------|---------|---------|
| **相遇** | 1-10章 | 男女主初次相遇，建立初始印象 | 冲突/误解/吸引力并存 |
| **发展** | 11-30章 | 关系逐步升温，经历考验 | 甜蜜→误会→和解→更甜蜜 |
| **危机** | 31-40章 | 最大障碍出现，关系面临破裂 | 虐心→分离→各自成长 |
| **和解** | 41-50章 | 克服障碍，关系升华 | 重逢→坦诚→承诺 |

**言情写作要点**：
- 男女主必须有独立的成长线，不能只是恋爱工具人
- 障碍必须来自性格/价值观/外部压力，不能是"我就是不说"式误会
- 甜蜜和虐心的比例建议7:3，虐心太久会流失读者

### 2.3 惊悚结构

| 阶段 | 章节占比 | 核心任务 | 紧张度 |
|------|---------|---------|--------|
| **危机** | 1-3章 | 引入威胁，建立恐惧 | 中→高 |
| **应对** | 4-10章 | 角色尝试应对，但步步失败 | 高→更高 |
| **追逐** | 11-20章 | 威胁升级，角色被逼入绝境 | 持续高位 |
| **逆转** | 21-25章 | 角色发现反击机会 | 高→极高 |
| **对抗** | 26-28章 | 最终对决 | 极高 |
| **结局** | 29-30章 | 威胁解除，但留下阴影 | 高→中 |

**惊悚写作要点**：
- 恐惧来源于"可知但不可控"——读者知道威胁存在但不知道何时出现
- 安全场景中埋设不安细节，让读者永远无法放松
- 结局可以不完全解决（留下心理阴影或暗示威胁未完全消除）

## 三、网文特色结构

### 3.1 退婚流模板

```
第1-3章：退婚羞辱 → 主角被当众退婚，受尽嘲讽
第4-6章：金手指觉醒 → 意外获得传承/系统/前世记忆
第7-15章：默默修炼 → 利用金手指快速成长，但隐藏实力
第16-20章：第一次打脸 → 在某个场合展示实力，震惊众人
第21-30章：离开新手村 → 进入更大世界，遇到更强对手
第31-50章：建立势力 → 收小弟/建宗门/夺资源
```

**关键爽点节点**：退婚对象后悔（第20章）、家族承认错误（第30章）、碾压当年羞辱者（第50章）

### 3.2 废材逆袭模板

```
第1-3章：被测出废材 → 灵根测试/武魂觉醒/资质检测垫底
第4-6章：意外获得传承 → 戒指里的老爷爷/上古功法/隐藏血脉
第7-20章：默默变强 → 表面废材，暗中修炼，扮猪吃虎
第21-25章：一鸣惊人 → 宗门大比/学院考核中展现实力
第26-40章：揭开身世 → 发现废材身份背后的秘密
```

**关键爽点节点**：第一次越级战斗（第15章）、宗门大比夺冠（第25章）、身世揭秘（第40章）

### 3.3 重生复仇模板

```
第1-3章：前世惨死 → 被背叛/被杀害，死不瞑目
第4-6章：重生归来 → 回到关键时间节点，带着前世记忆
第7-15章：改变命运 → 利用先知优势，避开前世陷阱
第16-25章：清算旧账 → 逐一报复前世仇人
第26-35章：发现更大阴谋 → 前世悲剧背后有更大黑手
第36-50章：最终对决 → 与幕后黑手决战
```

**关键爽点节点**：第一次改变命运（第10章）、仇人发现主角变了（第20章）、幕后黑手浮出水面（第35章）

### 3.4 系统流模板

```
第1-3章：系统降临 → 平凡人生中突然出现系统面板
第4-10章：新手任务 → 完成初始任务，获得奖励，适应系统
第11-25章：任务驱动成长 → 系统发布任务，主角被动或主动完成
第26-35章：系统升级 → 系统解锁新功能，任务难度升级
第36-45章：系统背后的秘密 → 发现系统不是凭空出现，有创造者和目的
第46-50章：超越系统 → 主角不再依赖系统，或与系统创造者对决
```

**关键爽点节点**：第一次使用系统能力（第5章）、系统升级（第26章）、系统秘密揭示（第40章）

### 3.5 穿越种田模板

```
第1-3章：穿越 → 穿越到古代/异世界，身份低微
第4-15章：生存立足 → 利用现代知识改善生活，逐步积累资源
第16-25章：扩大影响 → 从个人生存到影响村庄/城镇
第26-35章：势力碰撞 → 引起权贵注意，卷入更大格局
第36-50章：改变世界 → 用现代思维改造世界，成为一方势力
```

**关键爽点节点**：第一次用现代知识解决问题（第8章）、权贵被打脸（第25章）、建立自己的势力（第40章）

## 四、高级结构技巧

### 4.1 反转结构

```
前期埋线（第1-20章）：在正常叙事中埋设看似无关的细节
第一次小反转（第21章）：某个细节被重新解释，读者认知被颠覆
第二次中反转（第35章）：更深的真相被揭示，第一次反转只是冰山一角
第三次大反转（第48章）：终极真相，颠覆整个故事的解读
```

**反转设计原则**：
- 反转必须有前期伏笔支撑，不能凭空出现
- 每次反转后重读前文，读者应能发现"原来那些细节是这个意思"
- 反转的力度逐次递增，最后一次反转必须让读者拍案叫绝

### 4.2 多线叙事结构

| 叙事线 | 功能 | 交汇点 |
|--------|------|--------|
| **主线A**（主角当前行动） | 推进核心剧情 | — |
| **支线B**（配角/反派视角） | 提供信息差，制造悬念 | 每10-15章与主线交汇一次 |
| **暗线C**（历史/背景揭秘） | 逐步揭示世界观真相 | 每20-30章释放一个关键信息 |

**多线叙事原则**：
- 主线占比不低于70%，支线和暗线服务于主线
- 每次支线/暗线出现必须有明确目的（提供信息/制造悬念/塑造角色）
- 各线最终必须在结局处收敛，不能有未回收的叙事线

## 五、结构选择决策树

```
用户需求分析：
├── 有明确的金手指/系统设定？
│   ├── 是 → 系统流模板 或 退婚流模板
│   └── 否 → 继续判断
├── 主角初始状态？
│   ├── 废材/底层 → 废材逆袭模板
│   ├── 重生 → 重生复仇模板
│   └── 穿越 → 穿越种田模板
├── 核心情感线？
│   ├── 言情为主 → 言情结构
│   ├── 悬疑为主 → 悬疑结构
│   └── 惊悚为主 → 惊悚结构
└── 复杂世界观？
    ├── 是 → 三幕式 + 多线叙事
    └── 否 → 英雄之旅
```
