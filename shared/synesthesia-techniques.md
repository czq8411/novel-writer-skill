# 共享模块：通感写作技法指南

> **按需加载**：涉及场景描写、感官描写时加载  
> **全局规则**：遵循主控文件第二章核心铁律

---

## 一、五感体系

| 感官 | 核心体验 | 常见词汇 |
|------|---------|---------|
| 视觉 | 色彩、形状、光线 | 明亮、昏暗、鲜艳、模糊 |
| 听觉 | 声音、音调、节奏 | 清脆、低沉、嘈杂、寂静 |
| 嗅觉 | 气味、香气、异味 | 芬芳、刺鼻、清新、腐臭 |
| 味觉 | 味道、口感 | 甘甜、苦涩、辛辣、醇厚 |
| 触觉 | 触感、温度、重量 | 冰冷、温暖、粗糙、柔软 |

## 二、通感转换技法

| 转换类型 | 示例 | 效果 |
|---------|------|------|
| 视觉→听觉 | "月光像一首静谧的钢琴曲" | 赋予视觉画面以声音质感 |
| 听觉→视觉 | "他的笑声像金色的阳光洒落" | 赋予声音以视觉色彩 |
| 听觉→触觉 | "那声音冰冷刺骨，像一把刀划过皮肤" | 赋予声音以触感 |
| 嗅觉→视觉 | "茶香袅袅，在空气中绘出淡绿色的画卷" | 赋予气味以视觉形象 |
| 嗅觉→情感 | "故乡的泥土味，温暖得让人想哭" | 赋予气味以情感 |
| 味觉→听觉 | "她的歌声甜得像蜂蜜在舌尖融化" | 赋予声音以味道 |
| 触觉→听觉 | "丝绸划过肌肤，发出沙沙的低语" | 赋予触感以声音 |
| 情感→感官 | "悲伤像冰冷的雨水，浸透了他的骨头" | 赋予情感以感官体验 |

## 三、多感官联动描写

### 三感联动示例
```
厨房里，油锅滋滋作响（听觉），金黄色的油条在油锅里翻滚（视觉），一股浓郁的麦香混着油烟味飘了出来（嗅觉）。
```

### 五感全开示例
```
走进老巷子，青石板路在脚下发出咯吱的声响（听觉+触觉），两旁的墙壁爬满了翠绿的爬山虎（视觉），空气中弥漫着潮湿的泥土味和远处飘来的糖炒栗子香（嗅觉）。
```

### 感官对比示例
```
房间里静得可怕（听觉），只有窗外的蝉鸣像针一样扎进来（听觉→触觉），阳光透过窗帘的缝隙照进来，在地上投下斑驳的光影（视觉）。
```

## 四、通感写作禁忌

| 禁忌 | 错误示例 | 正确示例 |
|------|---------|---------|
| 过度堆砌 | 他的笑声像金色的阳光，又像甜蜜的蜂蜜... | 他的笑声像金色的阳光，温暖地洒在每个人身上。 |
| 逻辑混乱 | 她的歌声苦涩得像蓝色的眼泪。 | 她的歌声里带着一丝苦涩，像含着一颗没化的黄连。 |
| 比喻不当 | 他的话像一把红色的刀。 | 他的话像一把冰冷的刀，狠狠刺进她的心里。 |
| 喧宾夺主 | 战斗场景大量使用嗅觉描写 | 优先使用视觉和听觉描写战斗场景 |

## 五、题材专项指南

### 玄幻/仙侠
```
剑气纵横，发出刺耳的呼啸（听觉），金色的光芒撕裂空气（视觉），带着一股凛冽的寒气（触觉）。
```

### 都市言情
```
咖啡馆里，咖啡机发出咕噜咕噜的声响（听觉），拿铁的香气像一缕温柔的丝线（嗅觉→视觉），缠绕在两人之间。
```

### 悬疑惊悚
```
黑暗中，滴水声滴答作响（听觉），像一只无形的手，轻轻敲打着他的神经（触觉→情感），一股淡淡的血腥味在空气中弥漫（嗅觉）。
```

## 六、诊断与优化

| 检测项 | 标准 | 优化建议 |
|--------|------|---------|
| 感官覆盖 | 每段至少包含2种感官 | 增加其他感官描写 |
| 通感运用 | 每章至少1次通感转换 | 尝试通感写作 |
| 感官平衡 | 避免过度依赖某一种感官 | 平衡各感官描写 |
| 情感关联 | 感官描写应服务于情感表达 | 加强感官与情感的连接 |
