## 七、完整案例库

> 以下案例展示如何将题材指南应用到实际创作中。

### 案例1：玄幻·废材逆袭流

**题材**：玄幻/废材逆袭  
**核心情绪**：爽、燃  
**目标读者**：男性，18-25岁

#### 世界观设定（四维模板）

| 维度 | 设定内容 |
|------|---------|
| **社会结构** | 修炼者为尊，凡人如蝼蚁。宗门分为九品，一品最高。散修无资源，只能依附宗门或世家 |
| **经济体系** | 灵石为货币，分上中下三品。核心资源：灵石矿脉、天材地宝、功法秘籍。万宝商会垄断跨区域贸易 |
| **权力格局** | 三大一品宗门（天剑宗、万法宗、丹鼎宗）鼎立，皇朝居中调和。正邪对立，但灰色地带众多 |
| **文化习俗** | 强者为尊，弱肉强食。拜师礼极重，背叛师门为最大禁忌。每年祭天大典，每三年宗门大比 |

#### 开篇示例（黄金三章）

**第一章：觉醒**
```
林凡睁开眼睛的时候，发现自己躺在杂役房的草席上。

不是梦。

三天前，他还是林家嫡子，天赋测试为"地品灵根"，被天剑宗内定为真传弟子。
三天后，灵根被夺，修为尽废，被丢到杂役房等死。

"地品灵根？"林凡扯了扯嘴角，血从嘴角渗出来，"现在连凡品都不如了。"

他抬起手，看着掌心那道狰狞的疤痕——那是灵根被生生剥离时留下的。

疼吗？

疼。但比起心里的恨，这点疼算什么。

林凡闭上眼睛，脑海中响起三天前那个声音：

"检测到宿主生命垂危，至尊逆袭系统激活中..."

"激活进度：97%...98%...99%..."

"系统激活成功。"

"新手任务：在杂役房签到三天，奖励：天品灵根重塑丹。"

三天。只要再撑三天。

门外传来脚步声，还有那个熟悉到让他作呕的声音：

"林凡，林大少爷，该去倒夜香了。"

是赵虎，曾经跟在他身后摇尾乞怜的狗，现在成了踩他的人。

林凡深吸一口气，撑着身体坐起来。

三天。他只要三天。

三天后，谁是人谁是狗，还不一定。
```

**第二章：签到**
```
杂役房的夜，冷得像冰。

林凡蜷缩在草席上，听着外面呼啸的风声，在脑海中默念：

"系统，签到。"

"叮——杂役房签到成功，第1天。"
"奖励：基础炼体诀（残篇）"

一股暖流涌入四肢百骸，林凡感觉身体的疼痛减轻了一些。

他闭上眼睛，脑海中浮现出《基础炼体诀》的内容。

很基础的功法，但对此时的他来说，无异于救命稻草。

林凡开始按照功法运转体内那丝微弱的灵气。

一个周天。
两个周天。

直到天边泛起鱼肚白，他才停下来。

一夜修炼，效果微乎其微。但林凡能感觉到，自己的身体在好转。

"林凡！还不滚出来！"

赵虎的声音在门外响起，伴随着一脚踹在门板上。

林凡睁开眼睛，眼神平静。

他站起身，推开门。

赵虎站在门外，身后跟着两个杂役，都是曾经巴结过他的人。

"哟，林大少爷还能站起来？"赵虎嗤笑，"我还以为你昨晚就死了呢。"

林凡看着他，没有说话。

"看什么看？"赵虎被他的眼神看得有些发毛，恼羞成怒，"现在你是杂役，我是管事！我让你干什么你就得干什么！"

"今天你的任务是：清洗全院的夜香桶，然后去后山砍柴。"

"做不完，没饭吃。"

林凡点点头："好。"

他转身走向杂物间，去拿工具。

赵虎愣了一下，似乎没想到他会这么顺从。

"装什么装，"赵虎在身后骂道，"还以为自己是林家嫡子呢？现在你就是条狗！"

林凡脚步顿了一下，没有回头。

两天。再等两天。
```

**第三章：重塑**
```
第三天夜里，林凡躺在草席上，浑身是伤。

赵虎故意刁难，三天的工作量压在一天，他几乎累死在柴房。

但值了。

"系统，签到。"

"叮——杂役房签到成功，第3天。"
"恭喜宿主完成新手任务！"
"奖励：天品灵根重塑丹。"

一颗金色的丹药出现在林凡手中，散发着浓郁的药香。

林凡没有犹豫，一口吞下。

轰——

一股磅礴的药力在体内炸开，冲向四肢百骸。

疼！

比灵根被夺时还要疼！

林凡咬紧牙关，额头青筋暴起，指甲深深嵌入掌心。

但他没有叫出声。

不能叫。赵虎就在隔壁，不能让他发现异常。

药力在经脉中奔涌，所过之处，断裂的经脉重新连接，枯萎的丹田重新焕发生机。

最核心处，一道金色的光芒正在凝聚。

那是新的灵根。

比原来的地品更强！

天品灵根！

林凡能感觉到，天地灵气疯狂地涌入体内，速度是原来的十倍、百倍！

他运转《基础炼体诀》，灵气在经脉中奔流，修为节节攀升。

炼体一重。
炼体二重。
炼体三重。

一夜时间，连破三重！

天快亮的时候，林凡睁开眼睛。

两道精光从眸中射出，在黑暗中一闪而逝。

他站起身，活动了一下筋骨，浑身发出噼里啪啦的响声。

炼体三重，虽然还弱，但已经不是任人宰割的废物了。

门外传来脚步声，还有赵虎的骂声：

"林凡！滚出来！今天去..."

门开了。

林凡站在门口，居高临下地看着赵虎。

阳光从他身后照进来，在地上投下一道长长的影子。

赵虎愣住了。

三天前的林凡，眼神死寂，像条丧家之犬。

现在的林凡，眼神平静，却让他莫名地感到...恐惧。

"赵虎，"林凡开口，声音很轻，"这三天，多谢照顾。"

赵虎后退一步："你...你想干什么？"

"不想干什么。"林凡笑了笑，"只是想告诉你——"

"游戏结束了。"
```

### 案例2：言情·都市甜宠

**题材**：言情/都市甜宠  
**核心情绪**：甜、温馨  
**目标读者**：女性，18-30岁

#### 世界观设定（四维模板）

| 维度 | 设定内容 |
|------|---------|
| **社会结构** | 现代都市，职场阶层分明。主角是普通白领，男主是总裁/高管 |
| **经济体系** | 现代货币体系，房价高、生活成本高。职场收入差距大 |
| **权力格局** | 公司内部的权力斗争，男主处于权力顶端 |
| **文化习俗** | 现代都市文化，职场礼仪、社交规则。相亲、催婚等社会压力 |

#### 开篇示例（黄金三章）

**第一章：相亲**
```
苏晴看着对面坐着的男人，第无数次后悔答应老妈来相亲。

"苏小姐，"男人推了推眼镜，"我条件你也知道了，月薪一万二，有房有车。我对另一半的要求不高，婚后要辞职在家带孩子，孝顺我爸妈，还有..."

苏晴低头看了眼手机，闺蜜发来的消息："怎么样？"

她回："想死。"

"苏小姐？"男人皱眉，"你有没有在听？"

"有有有，"苏晴抬起头，露出职业假笑，"您继续说。"

"还有，婚后工资卡要上交给我妈，她管家有经验。你每个月我给你两千块零花钱，应该够用了吧？"

苏晴："..."

她深吸一口气，正准备找个借口开溜，身后突然传来一个声音：

"抱歉，我来晚了。"

苏晴愣了一下，转头看去。

一个男人站在她身后，穿着剪裁得体的西装，眉眼深邃，气场强大。

他看都没看对面的相亲男，直接对苏晴说："公司临时有事，等很久了吧？"

苏晴："？？？"

男人俯身，在她耳边低声说："配合我，我帮你解围。"

温热的气息拂过耳畔，苏晴耳朵一红。

对面的相亲男脸色变了："苏小姐，这位是？"

苏晴还没说话，男人已经直起身，淡淡地看了相亲男一眼：

"我是她男朋友。"

"另外，"他顿了顿，"月薪一万二就敢提这么多要求，谁给你的勇气？"

相亲男脸涨得通红："你..."

男人从钱包里抽出一张名片，放在桌上："如果有意见，可以让你的律师联系我。"

说完，他拉起苏晴的手："走吧，我请你吃饭。"

苏晴被他拉着走出咖啡厅，脑子还是懵的。

直到上了车，她才反应过来："等等，你是谁？"

男人发动车子，侧脸在夕阳下格外好看："陆景琛。"

"刚刚...谢谢你。"

"不客气。"陆景琛看了她一眼，"不过，我帮了你，你是不是也该帮我一个忙？"

"什么忙？"

"我奶奶催婚催得紧，"陆景琛语气平淡，"需要一个假女朋友应付一下。"

苏晴："..."

"为期三个月，报酬丰厚。"

苏晴想了想自己的房租，咬咬牙："成交。"

她没注意到，陆景琛嘴角勾起一抹得逞的笑。

——鱼儿上钩了。
```

**第二章：同居**
```
苏晴站在陆景琛的公寓门口，感觉自己像在做梦。

三天前，她还是一个普通的社畜，每天挤地铁、加班、被催婚。

三天后，她住进了总裁的豪宅，成了他的"假女友"。

"进来。"陆景琛打开门。

苏晴跟着进去，瞬间被眼前的景象震撼了。

落地窗、海景、极简风格的装修...这房子比她一年的工资还贵。

"你的房间在二楼，"陆景琛把钥匙递给她，"我住三楼，没事别上来。"

"哦..."

"厨房可以用，但别弄乱。"

"哦..."

"还有，"陆景琛顿了顿，"明天开始，你要跟我去公司。"

"啊？"

"假女友要演得像一点，"陆景琛面无表情，"我奶奶会派人监视。"

苏晴："...好的。"

她拖着行李箱上楼，找到自己的房间。

房间很大，带独立卫浴，比她租的出租屋好十倍。

苏晴扑到床上，滚了两圈。

好像...也不亏？

晚上，苏晴下楼找吃的。

陆景琛不在，她打开冰箱，里面只有水和几盒看起来就很贵的食材。

她不会做饭。

苏晴叹了口气，准备点外卖。

"不会做饭？"

身后突然传来声音，苏晴吓了一跳，差点把手机扔了。

陆景琛站在厨房门口，穿着居家服，头发还有些湿，显然是刚洗完澡。

居家服是灰色的，领口微微敞开，露出精致的锁骨。

苏晴："..."

她赶紧移开视线："我、我点外卖。"

"外卖不健康。"陆景琛走进厨房，"想吃什么？"

"啊？"

"我问你想吃什么，"陆景琛从冰箱里拿出食材，"我来做。"

苏晴瞪大眼睛："你会做饭？"

"很奇怪？"

"你是总裁啊！"

"总裁也要吃饭。"陆景琛开始切菜，动作熟练，"西红柿鸡蛋面可以吗？"

苏晴愣愣地点头。

她趴在厨房门口，看着陆景琛忙碌的背影。

灯光柔和，锅里的热气袅袅升起，陆景琛的侧脸专注而温柔。

苏晴突然有种...家的感觉。

"发什么呆？"陆景琛回头，"去洗手，马上好。"

"哦..."

苏晴转身去洗手，没注意到身后的男人嘴角微微上扬。

——第一步，成功。
```

**第三章：心动**
```
苏晴在陆景琛的公司待了一周，终于明白为什么他年薪千万了。

这男人简直是个工作机器。

早上七点起床，晚上十一点回家，中间除了吃饭上厕所，几乎都在工作。

苏晴作为"假女友"，每天的工作就是：在他办公室玩手机、陪他吃午饭、偶尔在他奶奶打电话来的时候说几句甜言蜜语。

轻松是轻松，但无聊也是真的无聊。

这天下午，苏晴在陆景琛办公室玩手机，突然听到外面传来争吵声。

"陆总，这个项目真的不能接！"

"理由？"

"对方公司信誉有问题，之前坑过好几家合作方..."

"我知道。"陆景琛的声音冷静，"但他们给出的条件，能让我们拿下东南亚市场。"

"可是..."

"没有可是。"陆景琛说，"去准备合同。"

苏晴皱了皱眉。

等助理出去，她忍不住说："那个项目有问题？"

陆景琛抬头看她一眼："听到了？"

"办公室隔音不好..."

"没事。"陆景琛低下头继续看文件，"商业决策，你不懂。"

苏晴："..."

她确实不懂商业，但她懂人心。

那个助理说得对，对方公司有问题。

晚上回家，苏晴翻来覆去睡不着。

凌晨一点，她爬起来，敲响了陆景琛的书房门。

"进来。"

陆景琛还在工作，桌上堆满了文件。

"还没睡？"他抬头看她。

"我..."苏晴咬了咬唇，"我想跟你说那个项目。"

陆景琛挑眉："嗯？"

"我不懂商业，但我查了一下，那家公司真的有问题。他们之前坑过的公司，最后都..."

"我知道。"陆景琛打断她。

苏晴愣住："你知道？"

"我知道他们有问题。"陆景琛靠在椅背上，"但我也需要那个市场。"

"可是..."

"苏晴，"陆景琛看着她，眼神深邃，"商业不是非黑即白。有时候，明知是坑也要跳，因为坑里有你想要的东西。"

苏晴沉默了。

她看着陆景琛，突然觉得他有点...孤独。

"那..."她小声说，"至少做好防护措施？"

陆景琛愣了一下，随即笑了："什么？"

"就是...合同里加一些保护条款？让对方先付定金？或者..."

她越说越小声，因为陆景琛看她的眼神越来越奇怪。

"怎么了？"苏晴紧张地问，"我说错了吗？"

"没有。"陆景琛站起身，走到她面前。

他比她高一个头，苏晴需要仰头才能看他的眼睛。

"苏晴，"陆景琛的声音很轻，"你为什么关心我？"

苏晴心跳漏了一拍。

"我...我们是合作关系啊，"她结结巴巴地说，"你亏了，我也拿不到报酬..."

"只是这样？"

"不然呢..."

陆景琛看着她，目光灼灼。

苏晴感觉自己的脸在发烧。

"早点睡。"陆景琛突然说，"明天还要陪我去见奶奶。"

"哦..."

苏晴转身要走，手腕却被拉住了。

"苏晴。"

"嗯？"

"谢谢。"陆景琛说，"很久没有人...这样关心我了。"

苏晴愣住。

她看着陆景琛的眼睛，那里面有一丝她看不懂的情绪。

"我..."

"去睡吧。"陆景琛松开她的手，"晚安。"

"晚安..."

苏晴走出书房，心跳如鼓。

她不知道的是，身后的陆景琛看着她的背影，眼神温柔得不像话。

——第二步，也成功了。
```

### 案例3：悬疑·刑侦破案

**题材**：悬疑/刑侦破案  
**核心情绪**：悬疑、紧张  
**目标读者**：男女通吃，20-35岁

#### 世界观设定（四维模板）

| 维度 | 设定内容 |
|------|---------|
| **社会结构** | 现代都市，警队内部有严格的等级制度。刑警、法医、痕检各司其职 |
| **经济体系** | 正常现代经济体系，案件往往涉及经济纠纷、遗产争夺等 |
| **权力格局** | 警队内部权力斗争，上级压力、媒体关注、舆论影响 |
| **文化习俗** | 法治社会，程序正义。但现实中存在灰色地带 |

#### 开篇示例（黄金三章）

**第一章：第三具尸体**
```
陈默数到第三具尸体时，发现了一个问题。

凶手少杀了一个人。

按照预告信上的数字，这里应该有四具，但冷藏柜里只有三具。

"陈队，"法医林悦走过来，"初步判断，死亡时间在48小时内。死因都是机械性窒息，但..."

"但什么？"

"第三具尸体的窒息痕迹和其他两具不一样。"林悦指着尸体的颈部，"前两具是绳索勒痕，这一具...像是被手掐死的。"

陈默皱眉："模仿作案？"

"或者是凶手临时改变了手法。"

陈默沉默片刻，走到冷藏柜前。

三具尸体，两男一女，都是二十多岁的年轻人。

他们的共同点：都是A大的学生，都参加过同一个社团——推理社。

"第四个人呢？"陈默问。

"还在找。"助手小王说，"根据社团名单，推理社有四名核心成员。这里三具尸体，还有一个叫...周晓雨的女生，目前失踪。"

"失踪？"

"嗯，已经失联48小时了。"

陈默眼神一凝。

如果周晓雨还活着，她可能是唯一的目击者。

也可能是...下一个受害者。

"找，"陈默说，"全城搜捕，一定要在她出事前找到她。"

"是！"

陈默走出法医室，点燃一支烟。

他抬头看着阴沉的天空，心里有种不好的预感。

这个案子，没那么简单。

预告信上的数字是"4"，但只死了三个人。

第四个人，是死是活？

如果是活，为什么凶手放过了她？

如果是死，尸体在哪里？

陈默吐出一口烟圈，眼神深邃。

游戏才刚刚开始。
```

**第二章：幸存者**
```
周晓雨被发现的时候，躲在A大图书馆的地下室里，已经三天没吃东西了。

"别过来！"她缩在角落里，手里握着一把美工刀，"别过来！"

"周晓雨，"陈默举起双手，示意自己没有威胁，"我是警察，我来救你。"

"警察..."周晓雨的眼神有些涣散，"真的是警察？"

"真的。"陈默慢慢靠近，"你看，这是我的证件。"

他掏出警官证，递过去。

周晓雨看了一眼，突然崩溃大哭："他们...他们都死了..."

陈默蹲下身，轻声说："我知道。你能告诉我发生了什么吗？"

"我们...我们在玩一个游戏..."周晓雨抽泣着，"推理社的传统，每年都要玩一次实景推理游戏..."

"什么游戏？"

"凶手游戏。"周晓雨抬起头，眼神恐惧，"一个人扮演凶手，其他人扮演侦探，在图书馆里找线索..."

"这次是谁扮演凶手？"

"是..."周晓雨突然顿住，脸色变得惨白。

"是谁？"

"是社长，"周晓雨的声音在颤抖，"李铭。"

陈默眼神一凝。

李铭。

三具尸体之一。

"你是说，扮演凶手的人，真的成了凶手？"

"我不知道..."周晓雨抱住头，"游戏开始后，我就躲起来了。我听到尖叫声，听到脚步声...我不敢出去..."

"你躲了三天？"

"嗯..."周晓雨点头，"我带了水和面包，本来准备躲到游戏结束..."

陈默沉默了。

如果周晓雨说的是真的，那李铭就是凶手。

但李铭已经死了。

死在冷藏柜里。

凶手死了，那谁杀了其他人？

"周晓雨，"陈默问，"除了你们四个，还有谁知道这个游戏？"

周晓雨愣了一下："应该...没有了吧？"

"仔细想想。"

周晓雨咬着唇，努力回想："社长说，这次的游戏很特别，他邀请了一个...一个神秘嘉宾。"

"神秘嘉宾？"

"嗯，社长说，这个嘉宾会扮演'真正的凶手'，给我们的游戏增加难度。"

陈默站起身，眼神冰冷。

神秘嘉宾。

真正的凶手。

这不是游戏，这是谋杀。

而那个神秘嘉宾，就是他们要抓的人。

"小王，"陈默对着对讲机说，"查一下李铭最近联系过的人，特别是...一个神秘嘉宾。"

"是！"

陈默看着蜷缩在角落里的周晓雨，心里有种预感。

这个案子，背后还有更大的阴谋。

而周晓雨，可能是唯一的突破口。

"带她回去，"陈默说，"保护好她，她是重要证人。"

"是！"

陈默走出地下室，阳光刺得他眯起眼睛。

他回头看了一眼黑暗的入口，心里默念：

等着我，我会抓住你的。
```

**第三章：真相浮现**
```
陈默在警局里待了整整一夜。

李铭的通讯记录、社交账号、银行流水...所有信息都被翻了个底朝天。

"陈队，"小王顶着黑眼圈走过来，"有重大发现。"

"说。"

"李铭在一个月前，收到了一笔五十万的转账。"

陈默抬头："谁转的？"

"一个空壳公司，查不到实际控制人。"小王顿了顿，"但是，我们在李铭的邮箱里发现了一封邮件。"

他递过平板电脑。

邮件内容很简单：

"想让你的推理社出名吗？按我说的做，酬劳五十万。"

发件人：匿名。

"李铭收了钱，"陈默说，"按照对方的指示，组织了这场'游戏'。"

"但他没想到，对方真的要杀人。"

"不，"陈默摇头，"他知道。"

小王愣住："什么？"

"他知道对方要杀人，"陈默站起身，走到白板前，"但他以为，死的会是周晓雨。"

他在白板上画了一个圈："周晓雨是推理社唯一的女生，家境普通，没有背景。如果她在'游戏'中意外死亡，没有人会怀疑。"

"但凶手改变了计划，杀了其他三个人，留下了周晓雨。"

"为什么？"

陈默沉默片刻："因为周晓雨看到了什么。"

他转身看向小王："周晓雨现在在哪？"

"在...在休息室。"

陈默快步走向休息室。

推开门，周晓雨坐在椅子上，手里捧着一杯热水。

看到陈默，她有些紧张："陈警官..."

"周晓雨，"陈默在她对面坐下，"我需要你再仔细回想一下，在地下室的那三天，你有没有看到或者听到什么特别的事情？"

周晓雨咬着唇："我...我真的没敢出去..."

"任何细节都可以，"陈默说，"比如，脚步声、说话声、甚至是...气味。"

周晓雨闭上眼睛，努力回想。

"我...我记得，"她突然说，"第二天晚上，我听到有人在说话。"

"说什么？"

"声音很小，我听不清...但是，"周晓雨睁开眼睛，"我听到了一个词。"

"什么词？"

"'审判'。"周晓雨说，"那个人说，'这是审判'。"

陈默眼神一凝。

审判。

这不是普通的谋杀，这是有预谋的、带有仪式感的杀人。

凶手不是在杀人，他是在...审判。

"还有吗？"陈默追问。

"还有..."周晓雨努力回想，"那个人的声音，我好像在哪里听过。"

"在哪里？"

"在..."周晓雨突然顿住，脸色变得惨白，"在推理社。"

陈默站起身："你是说，凶手是推理社的人？"

"不，不是社员..."周晓雨的声音在颤抖，"是...是指导老师。"

"谁？"

"张教授，"周晓雨说，"张明德教授。"

陈默立刻转向小王："查张明德，现在！"

"是！"

十分钟后，小王跑回来，脸色难看："陈队，张明德...失踪了。"

"什么时候？"

"三天前，也就是...案发当天。"

陈默握紧拳头。

张明德。

A大心理学教授，推理社的指导老师。

他为什么要杀自己的学生？

"查他的背景，"陈默说，"所有信息，我要知道他的一切。"

"是！"

陈默走到窗前，看着窗外的城市。

审判。

张明德在审判什么？

他的学生，犯了什么罪？

陈默有种预感，这个案子的真相，会比他想象的更加黑暗。

但他不会放弃。

无论真相多么残酷，他都要把它挖出来。

这是他的职责。

也是他对死者的承诺。
```
