### 市场分析三步法
1. **平台扫描**：列出目标平台（起点/番茄/七猫等）当前TOP50作品，分析热门标签
2. **读者画像**：定义核心读者群（年龄段/阅读场景/付费意愿），输出读者画像
3. **竞品分析**：选取3-5部同类作品，分析其爽点密度、更新频率、读者反馈

### 创意可行性评分表
| 维度 | 评分(1-5) | 标准 |
|------|:---:|------|
| 新颖度 | | 1=常见套路，3=微创新，5=前所未见 |
| 爽点密度 | | 1=稀少，3=适中，5=每章有爽点 |
| 市场匹配 | | 1=无对标，3=有对标，5=精准对标 |
| 持续性 | | 1=≤30章，3=≥100章，5=≥300章 |

综合分≥14/20 → 可行，10-13 → 需调整，<10 → 重新构思
