# 模式C子模块：内容扩充技巧

> 按需加载：章节字数不足、场景描写单薄、情节推进过快时加载本文件。
> 全局规则：遵循主控文件第二章核心铁律

---

## 一、诊断：为什么字数不足？

在扩充之前，先诊断字数不足的根本原因，对症下药：

| 症状 | 判断标准 | 根本原因 | 扩充方向 |
|------|---------|---------|---------|
| **场景太薄** | 场景切换频繁但每个场景描写不足100字 | 缺乏感官细节和空间感 | 场景肌理充实 |
| **跳跃太快** | 关键事件之间缺乏过渡，直接从A跳到C | 省略了中间步骤和角色反应 | 关键时刻放慢 |
| **情感平面** | 角色经历重大事件但无内心反应 | 缺乏内心世界展开 | 内心世界展开 |
| **事件稀少** | 全章只有1-2个事件 | 情节密度不足 | 次要情节穿插 |
| **对话干瘪** | 对话占比高但每段对话很短 | 缺乏潜台词和动作配合 | 对话层次丰富 |

## 二、防注水原则

扩充不等于注水。以下内容即使能增加字数也严禁使用：

| 注水类型 | 表现 | 为什么有害 |
|---------|------|-----------|
| **无意义风景描写** | "天空很蓝，白云朵朵，微风拂面……" | 与剧情无关，读者直接跳过 |
| **凑字数回忆** | 大段回忆前文已写过的内容 | 重复信息，浪费读者时间 |
| **重复已知信息** | 角色A告诉角色B读者已经知道的事 | 信息冗余，降低信息密度 |
| **过渡冗长** | "他走下楼，打开门，走到街上，拦了一辆车……" | 无冲突的日常动作不值得写 |
| **对话注水** | "你好""你好""吃了吗""吃了" | 无信息增量的对话是噪音 |

**扩充判断标准**：删掉这段扩充，故事是否变差？如果答案是否，说明扩充本身就是注水。

## 三、五大扩充技法

### 技法一：场景肌理充实

当场景描写过于单薄时，按感官层→空间层→情绪层逐层充实。

**操作步骤**：
1. 找到场景中只有视觉描写的段落
2. 补充至少1种非视觉感官（听觉/嗅觉/触觉）
3. 补充空间关系（光线方向、物体位置、空间大小感）
4. 通过环境细节的选择性描写映射人物情绪

**示例**：
```
扩充前（仅视觉）：
他走进山洞。洞里很暗，墙壁上有些壁画。

扩充后（感官层+空间层+情绪层）：
他弯腰钻进山洞。一股潮湿的霉味扑面而来，混着蝙蝠粪便的氨气。他打开手电——光柱在黑暗中切出一道窄窄的通道，照在洞壁上。

壁画从洞口一直延伸到深处。不是佛像，不是经文。是地图——密密麻麻的线条和标注，覆盖了每一寸石壁。他伸手触摸，颜料已经渗入石头纹理，至少有几百年了。

水滴从洞顶落下，在寂静中发出清脆的回声。一滴，两滴，三滴——节奏和他心跳刚好同步。
```

### 技法二：关键时刻放慢（子弹时间）

在剧情高潮处，将时间放慢，逐帧描写。这是增加字数的同时提升质量的最有效技法。

**适用场景**：
- 战斗决胜时刻（最后一击）
- 关键对话的转折点（某句话改变一切）
- 重大发现（发现关键线索或真相）
- 情感爆发（告白/决裂/重逢）

**操作步骤**：
1. 识别章节中的关键时刻（每章1-2个）
2. 将关键动作拆分为3-5个步骤
3. 每个步骤插入感官细节（视觉/听觉/触觉）
4. 插入角色的瞬间判断或联想
5. 放慢后接快节奏释放

**示例**：
```
扩充前（正常速度）：
林凡一剑刺出，正中对手胸口。对手倒下。

扩充后（子弹时间）：
林凡出剑。

这一剑他练了十年。从七岁开始，每天三千次，风雨无阻。剑尖刺破空气的声音他已经听过几百万次——但这一次不一样。

剑尖触及对手衣襟的瞬间，他看见了对手的眼睛。那双眼睛里没有恐惧，只有一种奇怪的释然。像一个人终于等到了等待已久的东西。

剑尖刺入。

林凡感觉到剑身传来的阻力——先是布料，然后是皮肤，然后是肌肉。每一层的触感都不同。布料是脆的，皮肤是韧的，肌肉是弹的。

他看见自己的手。握剑的手，指节发白。这只手和十年前第一次握剑时一模一样——但剑尖那头的人已经不一样了。

对手倒下的时候，嘴角带着笑。
```

#### 子弹时间进阶：动作分解五步法

将关键动作拆解为"准备→执行→意外→调整→完成"的微步骤序列，每一步穿插不同维度的描写。

| 步骤 | 描写维度 | 字数参考 | 示例关键词 |
|------|---------|---------|-----------|
| **准备** | 身体状态+环境感知 | 50-80字 | 呼吸、肌肉紧绷、视野收窄 |
| **执行** | 动作细节+感官放大 | 80-120字 | 速度感、阻力感、声音变化 |
| **意外** | 认知冲击+情感波动 | 60-100字 | 发现异常、瞬间判断、回忆闪现 |
| **调整** | 决策过程+身体响应 | 50-80字 | 改变策略、本能反应、代价意识 |
| **完成** | 结果呈现+余波感受 | 40-60字 | 冲击力、疼痛、寂静、反差 |

**五步法示例（武侠对决）**：

```
【准备】
沈惊鸿的右手在对方拔刀的瞬间已经按上了剑柄。他没有立刻拔——空间太窄，剑鞘太长，横着拔会碰到墙壁。他需要一个半转身的时间。呼吸在这一刻变得很慢，慢到他能听见自己心跳的每一次收缩。

【执行】
对方的第一刀是竖劈。沈惊鸿看见刀锋落下来的时候，大脑有一个极其清醒的瞬间：他判断出这一刀的角度偏右了，大约两寸。他侧身让过刀锋，右肩蹭到了墙面的粗石，火辣辣地疼了一下。第二刀紧跟而来，横扫。他不得不拔剑格挡。金属碰撞的声音在狭窄的巷道里像炸雷一样回荡。

【意外】
然后他看到了——对手收刀回拉的时候，左肋下有一个瞬间空档。不是很多，大约一息的时间。但那个空档的位置太完美了，完美到不像是失误。是陷阱？还是对方真的累了？

【调整】
他没有时间判断。剑尖已经找过去了——比大脑更快。他感受到剑身穿过布料、表皮、肌肉的不同阻力层次，像针穿过绸缎，然后是绸缎下面的东西。三寸左右。够了。再深一寸他的剑就拔不回来了。

【完成】
对方闷哼了一声，不是惨叫，是被疼痛闷住的那种声音，从鼻腔里挤出来的。刀从对方手中滑落，在青石板上弹了两下，声音清脆得像在敲钟。然后一切安静下来，只剩下两个人的喘息声。
```

#### 子弹时间进阶：思维展开法

在动作间隙插入角色的判断、回忆、预判和情感波动，将外部动作与内部世界交织。

**思维展开类型**：

| 类型 | 触发时机 | 展开内容 | 字数增量 |
|------|---------|---------|---------|
| **瞬间判断** | 动作执行前 | 对局势的快速评估 | +30-50字 |
| **感官联想** | 感官刺激时 | 由当前感官触发的记忆 | +40-80字 |
| **代价意识** | 做选择时 | 意识到选择的后果 | +30-60字 |
| **预判与修正** | 动作执行中 | 预期结果vs实际结果的对比 | +40-70字 |
| **情感闪回** | 关键时刻 | 与当前情境相关的情感记忆 | +50-100字 |

**思维展开示例（告白场景）**：

```
晚饭后的散步是江屿计划了两个星期的。他选的路线经过河边那棵老槐树——不是偶然，他提前来看过三次，确认那个位置够安静、路灯够暗、长椅够宽。【准备：筹划的细节】

但真正走到那棵树下的时候，他发现计划中的所有措辞都忘了。他停下来。不是从容的停步，是脚突然不听使唤的那种停。【执行：身体失控】

林栀又走了两步才发觉他没跟上来，回过头看他。河对岸的路灯光映在她脸上，明暗各半，她在暗处的那只眼睛里有疑问。【意外：对方的反应】

江屿张了张嘴，嗓子发紧。他打算说的话是一段经过反复推敲的告白——军训时她扎马尾的样子、图书馆她总坐第三排靠窗的位子——这些细节他排练过很多次。但此刻它们像一群受惊的鸟，在他脑子里乱飞。【思维展开：排练与现实的落差】

最后他说出来的话和排练的完全不同。声音比他预想的低很多："我喜欢你。从大一军训的时候就开始了。"【调整：放弃计划，本能开口】

说完之后的沉默是他人生中最长的三秒。林栀的表情没有变化——不是故意隐藏，是真的还没反应过来。他看到她的嘴唇微微张开，又合上，然后她低下了头。河风吹过来，把她垂下来的碎发拂到嘴角。她伸手把那缕头发别到耳后，这个动作比平时慢了很多。【完成：逐帧描写对方反应】
```

### 技法三：内心世界展开

当角色经历重大事件但缺乏内心反应时，展开内心世界。

**展开方式**：

| 方式 | 说明 | 示例 |
|------|------|------|
| **思维链** | 角色如何从A想到B再想到C | "他看见那枚戒指。和母亲手上那枚一模一样。但母亲说那枚戒指是独一无二的——是她外婆传下来的。那这一枚是……" |
| **感官触发** | 某个感官细节触发回忆 | "消毒水的气味。他立刻回到了七岁那年的病房。母亲的手正在变凉。" |
| **内心对话** | 角色内心的两个声音争论 | "杀了他。不，杀了他你就和他一样了。但他杀了你全家。杀了他你也回不去了。" |
| **身体反应** | 生理反应揭示心理状态 | "他的胃在收缩。不是紧张——是愤怒。每次愤怒到极点，他的胃就会先于大脑做出反应。" |

**展开原则**：
- 单次内心展开不超过3行（遵循模式C心理描写限制）
- 内心展开必须穿插动作或环境描写
- 内心展开的内容必须影响角色的后续行动

### 技法四：次要情节穿插

当章节事件稀少时，穿插次要情节线增加密度。

**穿插类型**：

| 类型 | 说明 | 字数增量 | 适用场景 |
|------|------|---------|---------|
| **配角视角** | 切换到配角视角，展示同一事件的不同侧面 | +200-400字 | 主线事件影响多人时 |
| **伏笔埋设** | 插入一个看似无关的细节，为后续剧情埋伏笔 | +100-200字 | 任何章节 |
| **环境变化** | 描写环境中的异常变化，暗示更大格局 | +100-200字 | 世界观宏大的作品 |
| **日常细节** | 插入角色的日常习惯，丰富人物形象 | +50-150字 | 节奏较慢的章节 |

**穿插原则**：
- 次要情节必须与主线有关联（直接或间接）
- 每次穿插不超过300字，避免喧宾夺主
- 穿插后必须回到主线，不能悬置

### 技法五：对话层次丰富

当对话过于功能化时，增加潜台词层和动作层。

**操作步骤**：
1. 找到纯功能对话段落（对话纯粹为了推进剧情）
2. 为每段对话增加潜台词层（角色说的≠角色想的）
3. 每3-4句对话插入动作/环境/心理描写
4. 用动作代替"他说/她说"标签

**示例**：
```
扩充前（功能对话）：
"你找到他了吗？"她问。
"找到了。"他说。
"在哪？"
"他已经死了。"

扩充后（潜台词+动作层）：
"你找到他了吗？"
她没抬头。手里的茶杯转了半圈，又转回来。
"找到了。"
他的声音很平。太平了——像在说今天的天气。
茶杯停住了。"在哪？"
他没有立刻回答。窗外有鸟叫，叫了三声。然后他说："他已经死了。"
茶杯掉在地上。碎成三片。她盯着碎片看了很久，然后弯腰去捡——手在发抖，捡了三次才捡起来。
```

## 四、扩充操作流程

```
1. 诊断字数不足原因（对照第一节五种症状）
2. 选择对应的扩充技法（可组合使用）
3. 定位需要扩充的具体段落
4. 执行扩充（每次扩充后检查是否超过3行心理描写限制）
5. 扩充后自检：删掉扩充内容，故事是否变差？
   - 是 → 扩充有效，保留
   - 否 → 扩充是注水，删除重做
```
