# 模式C子模块：章节开头技巧体系

> 按需加载：每章写作前加载本文件，选择最适合当前章节的开头技巧。
> 全局规则：遵循主控文件第二章核心铁律

---

## 核心原则

> 开篇即入戏：第一句必须是具体画面或动作，严禁背景铺垫和作者旁白。

每章开头是读者决定"继续读还是放下"的关键节点。好的开头在3秒内抓住注意力，差的开头让读者直接翻页。

---

## 十种强力开头技巧

### 技巧1：行动中开场（In Media Res）

直接从冲突或动作高潮开始，让读者立刻置身事中。

**技法要点**：
- 第一句就是动作或冲突
- 不解释前因后果，让读者在阅读中自行理解
- 适用于战斗章节、追逐章节、危机章节

**玄幻示例**：
```
剑尖距离林凡的咽喉只剩三寸。他没有退——身后是万丈深渊，身前是七柄指向他的剑。
```

**都市示例**：
```
陈默冲进急诊室的时候，手术灯已经亮了四十分钟。护士拦了他三次，他闯了三次。
```

**悬疑示例**：
```
老张踹开门的时候，尸体还有温度。凶手应该在三分钟之内离开——但整栋楼的监控没有拍到任何人。
```

### 技巧2：反常情境

呈现一个不符合常理的场景或细节，激发读者的好奇心。

**技法要点**：
- 呈现一个"不应该这样"的画面
- 不立即解释原因，让读者带着疑问继续阅读
- 适用于悬疑章节、转折章节、揭示章节

**玄幻示例**：
```
宗门大殿里跪着三十七个人。不是弟子——是长老。他们在等一个外门弟子来开会。
```

**都市示例**：
```
苏晚的办公桌上放着一封辞职信。是她自己的。但她不记得写过。
```

**悬疑示例**：
```
死者面带微笑。在一个被割喉的人脸上看到微笑，老张干了二十年刑警，第一次见。
```

### 技巧3：震撼对话

用一句惊人的对话开场，立刻建立冲突或悬念。

**技法要点**：
- 第一句对话必须有冲击力
- 对话内容暗示重大信息或冲突
- 适用于对峙章节、揭示章节、转折章节

**玄幻示例**：
```
"你师父是我杀的。"
林凡手里的茶杯停在半空。说话的人是他叫了十年师叔的人。
```

**都市示例**：
```
"孩子不是你的。"
陈默看着DNA报告，又看了看婴儿床里那双和他一模一样的眼睛。
```

**悬疑示例**：
```
"凶手自首了。"
"不可能。"
"他自己走进来的，身上还穿着作案时的衣服。"
老张放下电话。那件衣服三天前就该被烧掉了——他亲眼看着烧的。
```

### 技巧4：倒计时开场

从时间压力开始，制造紧迫感。

**技法要点**：
- 第一句就设定时间限制
- 时间限制必须有真实后果
- 适用于危机章节、营救章节、限时任务章节

**玄幻示例**：
```
还有一炷香。一炷香之后，护山大阵就会关闭，妖兽潮将涌入宗门。而林凡还在后山闭关，对这一切一无所知。
```

**都市示例**：
```
距离股市开盘还有四十分钟。陈默需要在四十分钟内凑齐三千万，否则他十年的心血将被强制平仓。
```

**悬疑示例**：
```
绑匪给了六个小时。现在已经过了五个半小时。老张看着地图上那个红圈——人质被藏在方圆十公里的某个地方，而他们还有三十分钟。
```

### 技巧5：重大发现

从发现关键线索、秘密或真相开始。

**技法要点**：
- 发现的内容必须改变角色对局势的认知
- 发现的呈现方式要有冲击力
- 适用于揭示章节、转折章节、解谜章节

**玄幻示例**：
```
林凡在藏经阁的暗格里找到了一封信。信纸泛黄，墨迹褪色，但落款的名字让他血液凝固——那是三百年前失踪的上一任掌门。
```

**都市示例**：
```
苏晚在未婚夫的手机里发现了一个隐藏相册。密码是她的生日。里面全是另一个女人的照片——从三年前到现在。
```

**悬疑示例**：
```
老张在死者的胃里找到了一把钥匙。钥匙上刻着一个地址——那是老张自己的家庭住址。
```

### 技巧6：危机时刻

从角色面临最大危机开始，立刻建立 stakes。

**技法要点**：
- 危机必须真实且紧迫
- 让读者立刻感受到"如果失败会怎样"
- 适用于战斗章节、逃亡章节、生死抉择章节

**玄幻示例**：
```
林凡的灵力在飞速流失。三分钟，最多三分钟，他就会从一个金丹期修士变成一个凡人——而追杀他的人就在洞府外面。
```

**都市示例**：
```
陈默的手机同时响起三个警报：公司账户被冻结、仓库被查封、合伙人的电话打不通。他看了一眼日期——距离新品发布会还有两天。
```

**悬疑示例**：
```
老张低头看着胸口的红点。狙击手。他站在犯罪现场的正中央，四面都是窗户，没有任何遮挡物。
```

### 技巧7：谜团浮现

从一个无法解释的现象开始，制造悬念。

**技法要点**：
- 现象必须足够离奇，让读者产生"为什么"的疑问
- 不立即解释，让谜团驱动阅读
- 适用于悬疑章节、世界观揭示章节、超自然事件章节

**玄幻示例**：
```
林凡醒来的时候，发现自己睡在宗门广场的正中央。周围围满了人。他不记得昨晚发生了什么——但他手里握着一把剑，剑上还有未干的血。
```

**都市示例**：
```
苏晚的手机收到一条短信，来自她自己的号码："离开他。你知道的还不够多。"
```

**悬疑示例**：
```
监控录像显示，死者走进电梯后就再也没有出来。但电梯门打开时，里面空无一人。三十二层楼，没有其他出口。
```

### 技巧8：背叛开场

从信任崩塌开始，立刻建立情感冲突。

**技法要点**：
- 背叛者必须是读者和主角都信任的人
- 背叛的呈现方式要有冲击力
- 适用于转折章节、黑化章节、复仇章节

**玄幻示例**：
```
林凡看着胸口穿出的剑尖。剑是从背后刺入的。他不用回头也知道握剑的人是谁——这把剑是他亲手锻造，送给她的。
```

**都市示例**：
```
陈默在会议室等了两个小时。最后等来的是合伙人的一条微信："公司我卖了。你的股份已经转让了。签名是你自己签的——三个月前那份'保险协议'，你记得吗？"
```

**悬疑示例**：
```
老张打开证物室的门。保险柜是空的。密码只有三个人知道：他、局长、和他搭档了十五年的老李。
```

### 技巧9：重大选择

从角色的艰难决定开始，立刻建立 stakes 和角色深度。

**技法要点**：
- 选择必须在两个都有代价的选项之间
- 选择的结果必须在后续剧情中产生实际影响
- 适用于角色成长章节、道德困境章节、关键转折章节

**玄幻示例**：
```
林凡面前摆着两样东西：一枚丹药，可以救他母亲的命；一把钥匙，可以打开困了他父亲十年的牢笼。他只能选一个。
```

**都市示例**：
```
陈默的手机上显示两个联系人：一个是能救他公司的投资人，一个是需要他出庭作证的前女友。两个电话，他只能接一个。另一个会永远挂断。
```

**悬疑示例**：
```
老张看着审讯室里的两个人。一个是他的线人，供出了关键情报；一个是他的线人的儿子，刚被抓到在案发现场。他可以选择相信情报抓儿子，也可以选择相信儿子废掉情报。
```

### 技巧10：结局预告

从未来关键时刻倒叙，制造"怎么会变成这样"的悬念。

**技法要点**：
- 预告的结局必须有冲击力
- 正文从"结局之前"开始，让读者带着"为什么会这样"阅读
- 适用于倒叙章节、回忆章节、命运感强的章节

**玄幻示例**：
```
林凡后来回想起这一天，才意识到从这一刻起，他就已经输了。不是输给对手——是输给了自己三年前做的一个决定。
```

**都市示例**：
```
如果陈默知道三天后他会亲手毁掉这一切，他也许不会在那个早晨推开那扇门。但他推了。
```

**悬疑示例**：
```
这是老张最后一次走进这间审讯室。四十分钟后，他会在这里做出一个决定——这个决定将结束他的警察生涯，但会拯救十三条人命。
```

## 开头技巧选择指南

| 章节类型 | 推荐技巧 | 次选技巧 |
|---------|---------|---------|
| 战斗/动作章节 | 技巧1（行动中开场）、技巧6（危机时刻） | 技巧4（倒计时） |
| 情感/关系章节 | 技巧8（背叛开场）、技巧9（重大选择） | 技巧3（震撼对话） |
| 悬疑/解谜章节 | 技巧7（谜团浮现）、技巧5（重大发现） | 技巧2（反常情境） |
| 转折/揭示章节 | 技巧5（重大发现）、技巧8（背叛开场） | 技巧10（结局预告） |
| 新卷/新地图 | 技巧2（反常情境）、技巧7（谜团浮现） | 技巧1（行动中开场） |
| 日常/过渡章节 | 技巧3（震撼对话）、技巧9（重大选择） | 技巧2（反常情境） |

## 开头禁忌

| 禁忌 | 说明 | 正确做法 |
|------|------|---------|
| 背景铺垫 | "在XX大陆的东方，有一个XX宗门……" | 从具体画面或动作开始 |
| 作者旁白 | "话说天下大势……" | 紧贴人物视角 |
| 天气开场 | "今天天气很好……" | 除非天气是剧情关键元素 |
| 人物介绍 | "林凡，男，18岁，XX宗门外门弟子……" | 通过行动和对话展示人物 |
| 世界观说明 | "这个世界分为九重天……" | 在剧情推进中自然揭示 |

## 十一、文学性开头进阶

> 本章节将 literary-mode.md 的叙事美学技法转化为可执行的开头策略。

### 限知视角开头
技巧：开头仅呈现角色当前感知范围的信息，刻意隐藏更大图景。

示例：
```
她不知道身后跟着什么——她只知道不能回头。脚步声在巷子里回荡，越来越近，但她分辨不出那是高跟鞋还是爪子。
```

### 不可靠叙述者开头
技巧：开头由一个有偏差/隐瞒/自我欺骗的叙述者讲述，在后续章节逐步揭露真实。

示例：
```
我不是那种人。我每个月按时交税，周末去养老院做义工，邻居家的猫丢了我会帮忙找。所以当警察第三次敲我的门时，我知道他们找错人了——他们不可能知道那件事。
```

### 环形结构开头
技巧：开头与结尾形成闭环，开头埋下一个意象/场景，结尾回到同一意象但已赋予新含义。

示例：
```
（第1章开头）他每天出门前都会看那盆绿萝一眼。还活着。他就放心了。
...
（第50章结尾）他推开门。绿萝枯了。他站在那里很久，然后笑了。也好，他终于不用每天确认还有东西活着了。
```
