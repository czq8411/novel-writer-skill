# 模式C子模块：创作风格指纹系统

> 按需加载：输入`风格指纹`查看当前风格，或输入`风格迁移 <目标风格>`调整风格。
> 全局规则：遵循主控文件第二章核心铁律

---

## 一、风格指纹提取

风格指纹是每位作者独特的写作DNA，通过多维度量化提取，实现风格一致性监控和风格迁移。

| 指纹维度 | 提取内容 | 量化方式 |
|---------|---------|---------|
| **句式指纹** | 短句(≤15字)/中句(16-30字)/长句(>30字)比例 | 百分比分布 |
| **词汇指纹** | 高频动词TOP20、高频形容词TOP20、独特用词 | 词频统计 |
| **节奏指纹** | 对话占比、描写占比、动作占比、心理占比 | 百分比分布 |
| **感官指纹** | 视觉/听觉/嗅觉/触觉/味觉调用频率 | 百分比分布 |
| **情绪指纹** | 情绪词密度、情绪波动幅度、情绪类型偏好 | 密度+分布 |
| **修辞指纹** | 比喻/排比/反问/设问使用频率 | 每千字次数 |

```
### 风格类型判定
**主风格**：冷峻动作流（辰东系）
**次风格**：留白意境流（烽火系）
**风格稳定性**：85%（近10章风格偏差<10%）
```

## 四、风格迁移

> 输入 `风格迁移 <目标风格>` 将当前章节调整为指定风格。

| 目标风格 | 代表作家 | 句式特征 | 词汇特征 | 节奏特征 |
|---------|---------|---------|---------|---------|
| **冷峻动作流** | 辰东 | 短句为主(50%+)，极少长句 | 动词密集，形容词克制 | 动作>对话>环境>心理 |
| **留白意境流** | 烽火戏诸侯 | 中短句交替，大量留白 | 诗意词汇，比喻丰富 | 环境≈对话>动作>心理 |
| **热血爽文流** | 天蚕土豆 | 中句为主，排比递进 | 情绪词密集，夸张修辞 | 动作>对话>心理>环境 |
| **细腻情感流** | 顾漫 | 中长句为主，细腻描写 | 情感词汇丰富，细节密集 | 心理≈对话>环境>动作 |
| **悬疑推理流** | 天下霸唱 | 中短句交替，信息密度高 | 名词具体，动词精准 | 对话>环境>动作>心理 |

### 风格迁移报告

```markdown
## 风格迁移：当前风格 → 冷峻动作流

### 调整项
| 维度 | 调整前 | 调整后 | 变化 |
|------|--------|--------|------|
| 短句占比 | 35% | 52% | +17% |
| 形容词密度 | 8.5/千字 | 4.2/千字 | -51% |
| 心理描写占比 | 18% | 8% | -10% |
| 动作描写占比 | 25% | 42% | +17% |

### 迁移后风格匹配度：88%
```

## 五、风格演化追踪

> 每10章自动生成风格演化报告，追踪创作风格的长期变化趋势。

```markdown
## 风格演化追踪（第1-50章）

### 句式演化
```
第1-10章：  短45% 中35% 长20%
第11-20章： 短42% 中38% 长20%
第21-30章： 短40% 中40% 长20%
第31-40章： 短38% 中42% 长20%
第41-50章： 短35% 中45% 长20%
趋势：短句占比逐渐下降，中句占比上升
```

### 风格演化洞察
- 句式从冷峻简洁向细腻丰富过渡
- 心理描写占比从8%上升至15%
- 感官调用从视觉为主向多感官均衡发展
- 整体风格从"辰东系"向"猫腻系"演化
```
