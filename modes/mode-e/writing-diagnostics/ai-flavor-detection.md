# 模式C子模块：AI味检测系统

> 按需加载：输入`检测AI味`触发全面检测，或输入`句式检查`/`五感检查`等单项检测。
> 全局规则：遵循主控文件第二章核心铁律

---

## 一、AI味检测概述

AI味是指AI生成文本中常见的机械化、套路化、缺乏人性化的特征。本系统通过多维度量化检测，识别并消除AI味，提升文本质量。

### 1.1 AI味核心特征

| 特征类型 | 具体表现 | 严重程度 |
|---------|---------|---------|
| **句式单一** | 主谓宾结构占比过高，缺乏句式变化 | 高 |
| **情感机械** | 情绪变化线性单一，缺乏复杂性和矛盾性 | 高 |
| **套路描写** | 使用 cliché 比喻和常见表达 | 中 |
| **五感失衡** | 过度依赖视觉，其他感官缺失 | 高 |
| **逻辑过清** | 人物思维过于清晰，缺乏真实人类的混乱感 | 中 |
| **词汇平庸** | 缺乏独特用词，词汇选择保守 | 中 |

## 二、句式多样性检测

### 2.1 检测标准

**健康指标**：
- 主谓宾结构占比 ≤ 50%
- 变异句式（倒装、省略、被动、从句等）占比 ≥ 30%
- 单句长度变化系数 ≥ 0.4

### 2.2 句式类型库

| 句式类型 | 示例 | 使用场景 |
|---------|------|---------|
| **主谓宾** | 他走进了房间。 | 基础叙述 |
| **倒装句** | 走进房间时，他停下了脚步。 | 强调动作 |
| **省略句** | 脚步很轻。 | 营造氛围 |
| **被动句** | 门被缓缓推开。 | 悬疑/未知 |
| **从句嵌套** | 当他走进那个充满血腥味的房间时，他意识到事情不对劲。 | 复杂场景 |
| **对话打断** | "你——"他顿了顿，"你怎么在这里？" | 情绪转折 |
| **感官前置** | 一股腐臭味扑面而来。 | 氛围营造 |

### 2.3 句式优化示例

**原文（AI味重）**：
```
他走进了房间。他看到了尸体。他蹲下身检查。他发现了一些线索。
```

**优化后（句式多样）**：
```
推开门的瞬间，一股腐臭味扑面而来。

房间中央，尸体仰面倒在血泊中，四肢扭曲成诡异的角度。

蹲下身时，膝盖发出轻微的脆响。指尖触到死者手腕的瞬间，一股寒意顺着指尖爬上来——尸体已经凉了，死亡时间至少在六小时以上。
```

### 2.4 句式检查清单

- [ ] 连续5句不以相同结构开头
- [ ] 每段至少有1个变异句式
- [ ] 对话和叙述交替出现，避免大段单一类型
- [ ] 关键场景使用倒装/省略营造氛围

## 三、五感平衡检测

### 3.1 五感分布标准

**健康分布**：
```
视觉: 50-60% ████████████████████████████████
听觉: 15-20% ██████████
触觉: 10-15% ███████
嗅觉: 8-12%  █████
味觉: 3-5%   ██
```

**警告阈值**：
- 视觉 > 70%：严重失衡
- 触觉 < 5%：体感缺失
- 嗅觉 < 3%：氛围不足

### 3.2 五感描写词库

| 感官 | 常用词汇 | 高级词汇 |
|------|---------|---------|
| **视觉** | 看见、红色、明亮 | 斑驳、氤氲、幽暗、刺目 |
| **听觉** | 听到、声音、响 | 嗡鸣、窸窣、轰鸣、死寂 |
| **触觉** | 摸到、冷、热 | 刺痛、黏腻、粗糙、战栗 |
| **嗅觉** | 闻到、香、臭 | 腥甜、腐臭、清冽、霉味 |
| **味觉** | 尝到、甜、苦 | 铁锈味、酸涩、甘冽、腥咸 |

### 3.3 五感优化示例

**原文（视觉主导）**：
```
他走进了房间，看到了尸体。尸体脸色苍白，眼睛睁得很大。他蹲下身检查，发现死者手里握着什么东西。
```

**优化后（五感平衡）**：
```
推开门的瞬间，一股腐臭味混合着血腥味扑面而来（嗅觉）。

房间很暗，只有窗外透进来的微弱光线照亮了中央那具尸体（视觉）。死者脸色苍白，眼睛睁得很大，瞳孔扩散成诡异的灰白色。

他蹲下身，膝盖触到冰冷的地砖（触觉）。指尖碰到死者手腕的瞬间，一股寒意顺着指尖爬上来——尸体已经凉了。耳边传来窗外风吹过树叶的沙沙声，在寂静的房间里格外清晰（听觉）。

他掰开死者僵硬的手指，发现手里握着一枚铜钱。铜钱上沾着暗红色的血迹，凑近闻了闻，除了血腥味，还有一股淡淡的檀香味（嗅觉）。
```

### 3.4 五感检查清单

- [ ] 每个关键场景至少包含3种感官描写
- [ ] 视觉占比不超过70%
- [ ] 触觉/嗅觉在氛围营造中至少出现1次
- [ ] 避免"他看到/听到/闻到"等直白表达

## 四、情感表达层次检测

### 4.1 情感表达标准

**健康指标**：
- 单一情绪持续时间 ≤ 200字
- 混合情绪（矛盾/复杂）占比 ≥ 30%
- 情绪变化有明确触发点

### 4.2 情感层次模型

```
表层情绪 → 深层情绪 → 潜意识情绪
   ↓           ↓            ↓
（可观察）  （可感知）   （隐含）
```

**示例**：
```
表层：沈默面无表情地看着尸体（冷静/专业）
深层：他的手指微微颤抖（震惊/恐惧）
潜意识：那个画面让他想起了三年前（创伤/自责）
```

### 4.3 情感优化示例

**原文（情感单一）**：
```
沈默很震惊。他想不通为什么妹妹会出现在这里。他很担心妹妹的安全。
```

**优化后（情感层次）**：
```
沈默盯着屏幕，脸上的表情没有变化——至少看起来没有。

但林晚注意到，他的手指在微微颤抖。那只握着鼠标的手，指节因为用力而发白。

"沈顾问？"她轻声问。

沈默没有回答。他的视线还停留在那个画面上：沈雨抬起头，朝摄像头的方向看了一眼。那一眼，让沈默的血液瞬间凝固。

三年了。

一千多个日夜，他无数次在梦里见到这个画面。有时是在人群中擦肩而过，有时是在街角的咖啡馆，有时是在这样的案发现场。每次醒来，都是空。

但这次不是梦。

"查。"沈默的声音很沙哑，像是从喉咙深处挤出来的，"查那栋楼的监控，找到那个女人。"

林晚想再问什么，但看到沈默的侧脸，她闭上了嘴。

那张脸上没有任何表情，像是一尊雕塑。但林晚注意到，他的眼眶红了。
```

### 4.4 情感检查清单

- [ ] 避免连续使用单一情绪词（如"他很害怕"重复多次）
- [ ] 通过动作/细节暗示情绪，而非直白陈述
- [ ] 展示情绪的矛盾性（如"既希望是她，又害怕是她"）
- [ ] 情绪变化有明确的触发事件

## 五、Cliché 检测与替换

### 5.1 常见 Cliché 库

| Cliché 表达 | 问题 | 替换建议 |
|------------|------|---------|
| "血液瞬间凝固" | 过度使用 | "一股寒意从脊椎爬上来" |
| "眼泪无声地滑落" | 常见表达 | "眼眶发热，但他没眨眼" |
| "像是一头独自舔舐伤口的狼" | 过度使用的比喻 | "肩膀微微佝偻，脚步沉重" |
| "时间仿佛静止了" | 常见表达 | "秒针的滴答声突然变得震耳欲聋" |
| "心中五味杂陈" | 抽象表达 | 具体描述矛盾的情绪表现 |
| "他的眼神像刀子一样锋利" | 常见比喻 | "眼神冷得像冰，刺得人皮肤发疼" |

### 5.2 比喻创新原则

1. **具体化**：避免抽象比喻，使用具体可感的意象
2. **个性化**：结合角色身份和场景特点创造独特比喻
3. **多感官**：比喻可以涉及任何感官，不限于视觉

**示例**：
```
原文: "他的眼神像刀子一样锋利。"

优化（刑侦背景）: "他的眼神像是X光，能穿透皮肤看到骨头。"

优化（灵气复苏背景）: "他的眼神像是某种探测法术，让人有种被从内到外扫描的不适感。"
```

## 六、AI味综合评分

### 6.1 评分维度

| 维度 | 权重 | 检测方法 | 健康阈值 |
|------|------|---------|---------|
| 句式多样性 | 25% | 主谓宾结构占比 | ≤ 50% |
| 五感平衡 | 20% | 视觉占比 | ≤ 70% |
| 情感层次 | 20% | 单一情绪持续时间 | ≤ 200字 |
| Cliché 密度 | 15% | 每千字 cliché 数量 | ≤ 1个 |
| 词汇独特性 | 15% | 高频词占比 | ≤ 30% |
| 逻辑混乱度 | 5% | 人物思维跳跃性 | ≥ 10% |

### 6.2 评分等级

| 分数 | 等级 | 说明 | 处理建议 |
|------|------|------|---------|
| 0-3 | 优秀 | 几乎无AI味，接近人类写作 | 保持 |
| 3-5 | 良好 | 轻微AI味，不影响阅读 | 微调 |
| 5-7 | 中等 | 明显AI味，需要优化 | 重点修改 |
| 7-9 | 严重 | 重度AI味，影响阅读体验 | 大幅重写 |
| 9-10 | 极重 | 典型AI文本，需完全重构 | 重新生成 |

### 6.3 检测报告格式

```markdown
## AI味检测报告

### 总体评分: 6.8/10 (中等AI味)

### 分项评分
| 维度 | 得分 | 状态 | 说明 |
|------|------|------|------|
| 句式多样性 | 7.5 | 🔴 | 主谓宾结构占比60%，需增加变异句式 |
| 五感平衡 | 7.0 | 🔴 | 视觉占比78%，需增加其他感官 |
| 情感层次 | 6.5 | 🟡 | 存在200字以上的单一情绪段落 |
| Cliché 密度 | 4.0 | 🟢 | 每千字2个，在可接受范围 |
| 词汇独特性 | 6.0 | 🟡 | 高频词占比35%，需增加词汇多样性 |
| 逻辑混乱度 | 2.0 | 🔴 | 人物思维过于清晰，缺乏混乱感 |

### 主要问题
1. **句式单一**: 连续使用"他..."开头的主谓宾结构
2. **五感失衡**: 视觉占比过高，触觉/味觉缺失
3. **情感机械**: 情绪变化过于线性，缺乏复杂性

### 优化建议
1. 将主谓宾结构占比降至50%以下
2. 每个关键场景增加至少2种非视觉感官描写
3. 展示情绪的矛盾性和复杂性
```

## 七、AI味消除技巧

### 7.1 写作前自检

在开始写作前，默念以下原则：
1. **"我不是在说明，我是在呈现"** —— 用场景和细节代替直白陈述
2. **"人物不是逻辑的，是混乱的"** —— 展示真实人类的矛盾和混乱
3. **"读者的身体也要参与"** —— 调动所有感官，不只是视觉
4. **"避免第一个想到的表达"** —— 寻找更新颖的比喻和描述

### 7.2 写作中检查

每完成300字，暂停检查：
- [ ] 这300字中有多少种句式？
- [ ] 涉及哪些感官？视觉占比是否过高？
- [ ] 情绪是否单一？是否有矛盾性？
- [ ] 有没有使用 cliché？

### 7.3 写作后优化

完成章节后，执行以下优化：
1. **句式优化**：将连续的主谓宾结构替换为变异句式
2. **五感补充**：在视觉主导的场景中插入其他感官描写
3. **情感深化**：为单一情绪增加矛盾性和复杂性
4. **Cliché 替换**：替换所有常见的比喻和表达
