## 诊断维度

### 前置分析
在生成报告前，请先读取前3-5章细纲和正文，并分析：
- **当前阶段**：是开篇（1-20章）、发展期（21-80章）还是高潮期（80章以后）？
- **核心矛盾**：当前的主线任务是什么？
- **最近断章**：前3-5章的结尾是否留有钩子？本章是否能有效承接？
- **逻辑自查**：最近剧情是否违背了"逻辑第一原则"（如：金手指无代价、反派降智）？
- **章节充实度初判**：快速扫描本章字数、事件数、信息点数，初步判断是否存在内容单薄风险
- **大纲关联度**：本章涉及的大纲模块有哪些？大纲中对应模块的详细程度是否足以支撑本章描写？

### 五-A、人物关系过渡诊断

> 检测主要角色之间的关系发展是否有合理的过渡章节。关系跳跃（如"陌生人→兄弟"在1章内完成）是读者体验的隐形杀手。

#### 关系发展阶段定义

| 阶段 | 代码 | 典型表现 | 最少过渡章节数 |
|------|:---:|---------|:---:|
| **陌生** | S0 | 互不认识或仅一面之缘 | — |
| **注意** | S1 | 一方注意到另一方的特殊之处 | 1章 |
| **接触** | S2 | 首次对话/互动/合作 | 1章（从S1起） |
| **熟悉** | S3 | 多次互动，建立基本信任 | 2章（从S2起） |
| **信任** | S4 | 愿意分享秘密/托付后背 | 2章（从S3起） |
| **羁绊** | S5 | 生死之交/不可替代 | 3章（从S4起） |

#### 检测规则

| 检测项 | 规则 | 级别 |
|--------|------|:--:|
| **关系跳跃** | 相邻章节关系阶段跨越>2级（如S0→S3）且中间无互动描写 | P0 |
| **关系突变** | 相邻章节关系阶段跨越=2级（如S0→S2），检查是否有≥300字的互动描写 | P1 |
| **关系倒置** | 关系阶段出现倒退（S3→S1）且无合理解释（背叛/误会） | P0 |
| **关系遗忘** | 已建立S3+关系的角色连续5章无互动 | P1 |
| **单向关系** | A对B的关系阶段与B对A的关系阶段差距>2级且未说明原因 | P2 |
| **关系升级无触发事件** | 关系升级（如S2→S3）但对应章节无共同经历事件（战斗/危机/分享秘密） | P1 |

#### 关系过渡评分

| 关系对 | 起始章节 | 起始阶段 | 当前章节 | 当前阶段 | 过渡章节数 | 是否合理 | 评分 |
|--------|:---:|:---:|:---:|:---:|:---:|:---:|:---:|
| 主角↔角色A | 第X章 | S0 | 第Y章 | S4 | N章 | ❌/⚠️/✅ | <<1-10>> |

### 八、读者视角模拟

> 跳出作者视角，以读者身份审视每一章。读者不会关心你的大纲有多精妙，他们只关心"这一章好不好看"。

#### 期待值曲线
模拟读者在每章结束时的期待值变化（1-10分）：

| 章节 | 开篇期待 | 过程峰值 | 章末期待 | 变化趋势 | 风险信号 |
|------|---------|---------|---------|---------|---------|
| 第X章 | <<1-10>> | <<1-10>> | <<1-10>> | <<↑/↓/→>> | <<无/连续下降/低于5分>> |

**期待值管理原则**：
- 章末期待值应≥开篇期待值（读者读完应该更想看下一章）
- 连续3章期待值下降→读者弃书高风险
- 章末期待值<5→必须强化钩子

#### 弃书风险点
标注可能导致读者弃书的节点：

| 风险类型 | 触发条件 | 当前章节 | 风险等级 | 修复建议 |
|---------|---------|---------|---------|---------|
| 节奏拖沓 | 连续3章无实质剧情推进 | 第X章 | 🔴高 | 插入冲突/反转/新信息 |
| 主角憋屈 | 主角连续受挫无反击 | 第X章 | 🔴高 | 安排反击或伏笔暗示即将反击 |
| 逻辑硬伤 | 出现无法解释的剧情漏洞 | 第X章 | 🔴高 | 补充解释或修改剧情 |
| 审美疲劳 | 连续5章同类型爽点 | 第X章 | 🟠中 | 更换爽点类型或插入支线 |
| 信息过载 | 单章新信息>5个核心点 | 第X章 | 🟠中 | 拆分信息到后续章节 |

#### 爽点满足度
评估每个爽点是否达到预期强度：

| 爽点位置 | 设计类型 | 预期强度 | 实际强度 | 满足度 | 不足原因 |
|---------|---------|---------|---------|--------|---------|
| 第X章 | <<打脸/升级/获得>> | <<1-10>> | <<1-10>> | <<✅/⚠️/❌>> | <<铺垫不足/反派太弱/反转不够>> |

#### 悬念有效性
评估钩子是否真正引发"下一章会怎样"的好奇心：

| 钩子位置 | 钩子类型 | 悬念强度 | 是否有效 | 问题诊断 |
|---------|---------|---------|---------|---------|
| 第X章末 | <<悬念式/危机式/预告式/登场式>> | <<1-10>> | <<✅/⚠️/❌>> | <<答案太明显/与主线无关/读者不关心>> |

**悬念有效性检验**：
- 读者能否用"是/否"回答这个悬念？→ 能则太弱，需要开放式悬念
- 这个悬念的答案是否在下一章立即揭晓？→ 是则太浅，需要延迟满足
- 读者是否关心这个悬念的答案？→ 不关心则需要加强情感绑定

### 十、神秘感与悬念层次诊断

> 爆款小说的核心驱动力是"读者想知道接下来会发生什么"。过早揭露终极反派、一次性解答所有谜题、缺乏多层次悬念，是读者流失的首要原因。

#### 悬念层次模型

| 层次 | 类型 | 揭晓时机 | 示例 |
|:---:|------|:---:|------|
| **L1-终极悬念** | 最终boss身份/世界真相/主角终极命运 | 最后一卷 | 诡秘之主中"远古太阳神"的真相 |
| **L2-卷级悬念** | 本卷核心冲突的结局/关键角色的真实身份 | 本卷末尾 | 斗破苍穹中"三年之约"的结果 |
| **L3-段级悬念** | 当前副本/事件的谜底 | 10-30章内 | 当前秘境中隐藏的宝物是什么 |
| **L4-章级悬念** | 章末钩子/下一章的答案 | 下一章 | "门后站着的人是谁？" |

#### 检测规则

| 检测项 | 规则 | 级别 |
|--------|------|:--:|
| **终极反派过早揭露** | 前20章内直接点名或详细描述最终boss的身份/能力/动机 | P0 |
| **悬念层次缺失** | 当前章节范围内，L1-L4悬念层次<3层（即读者没有足够多的"想知道"） | P0 |
| **谜题一次性解答** | 单章内同时解答≥3个之前设置的悬念（"大揭秘综合征"） | P1 |
| **悬念无递进** | 连续5章悬念强度无升级（始终是同一层次的疑问） | P1 |
| **假悬念/无效悬念** | 设置的悬念答案过于明显（读者能轻易猜到）或与主线无关 | P2 |
| **神秘感透支** | 前30章已揭示的世界观信息占大纲总设定的>60% | P0 |
| **反派扁平化** | 前50章内所有反派均为一次性工具人，无层次化反派体系 | P1 |
| **幕后黑手缺失** | 前50章无任何暗示存在更高层次的幕后势力/人物 | P1 |

#### 神秘感健康度评分

| 检查项 | 评分标准 |
|--------|---------|
| 终极boss隐藏度 | 完全未提及(10分)/仅侧面暗示(8分)/提及称号未描述(5分)/直接描述(2分)/详细揭露(0分) |
| 悬念层次丰富度 | 4层齐全(10分)/3层(7分)/2层(4分)/1层(1分) |
| 谜题解答节奏 | 逐层递进(10分)/偶尔集中(6分)/频繁一次性解答(2分) |
| 世界观信息保留率 | 揭示<30%(10分)/30-50%(7分)/50-70%(4分)/>70%(1分) |
| 反派层次感 | 3层以上反派体系(10分)/2层(7分)/1层(4分)/全一次性(1分) |

```markdown
|:---:|------|:---:|------|:---:|
```

### 十一、世界观展开节奏诊断

> 读者需要通过剧情自然了解世界，而非被灌输设定。宗门大比之前应对宗门有充分介绍，重大事件前应有足够的背景铺垫。

#### 世界观展开阶段模型

| 阶段 | 章节范围 | 应完成的世界观展开 | 禁忌 |
|------|:---:|------|------|
| **引入期** | 1-10章 | 主角所在小环境（家族/村落/入门测试场所） | 禁止展开全国/全大陆地图 |
| **探索期** | 11-30章 | 首个大场景全貌（宗门/学院/城市）、基本规则体系 | 禁止引入其他大陆/位面 |
| **扩展期** | 31-60章 | 势力分布、周边地域、修炼体系全貌 | 禁止展开全部高阶战力 |
| **深化期** | 61-100章 | 历史背景、隐藏势力、世界真相的暗示 | 可逐步展开更大世界 |

#### 检测规则

| 检测项 | 规则 | 级别 |
|--------|------|:--:|
| **重大事件前置铺垫不足** | 宗门大比/秘境开启/大战等重大事件前，相关场景/规则/人物的介绍章节<3章 | P0 |
| **场景跳跃过快** | 主角在5章内跨越≥3个不同大地域（如家族→宗门→秘境→王都） | P0 |
| **宗门/组织介绍敷衍** | 主角加入宗门后，宗门结构（长老/峰主/各堂）、规则（门规/贡献点）、势力派系介绍<500字 | P0 |
| **赛事/大比信息缺失** | 宗门大比/比武招亲等赛事开始前，未交代：参赛资格、赛制、奖励、历史背景 | P0 |
| **世界观信息倾倒** | 单章内通过旁白/系统说明连续输出世界观信息>800字（"设定说明书"式写作） | P1 |
| **地图/地理缺失** | 主角活动范围已跨越≥3个地点，但从未描述过各地之间的空间关系 | P1 |
| **日常细节缺失** | 连续10章无任何日常生活描写（衣食住行/修炼日常/人际交往） | P1 |
| **规则体系不一致** | 同一规则在不同章节有矛盾描述（如贡献点兑换比例前后不一） | P0 |

#### 世界观展开评分

| 检查项 | 评分标准 |
|--------|---------|
| 场景展开节奏 | 逐层递进(10分)/偶尔跳跃(6分)/频繁跳跃(2分) |
| 重大事件铺垫 | 充分铺垫≥3章(10分)/基本铺垫1-2章(6分)/无铺垫(1分) |
| 宗门/组织描写 | 详细立体(10分)/基本介绍(6分)/一笔带过(2分) |
| 信息输出方式 | 剧情中自然展现(10分)/偶尔灌输(6分)/大量旁白灌输(2分) |
| 日常细节丰富度 | 丰富自然(10分)/偶有提及(6分)/完全缺失(2分) |

#### 诊断输出格式

```markdown

#### 展开阶段判定

- **应处阶段**：<引入期/探索期/扩展期/深化期]
- **实际展开度**：<超前/正常/滞后]

| 事件 | 发生章节 | 前置铺垫章节数 | 铺垫内容 | 是否充分 |
|------|:---:|:---:|------|:---:|
| 宗门大比 | 第X章 | N章 | <<铺垫内容>> | <<✅/❌>> |
```

### 十二、人物群像深度诊断

> 爆款小说的人物不是工具人，每个主要角色都应该有独立的动机、背景和成长弧线。读者记住的是人物，不是情节。

#### 人物深度维度

| 维度 | 说明 | 最低标准 |
|------|------|---------|
| **独立动机** | 角色自己的目标/追求（不只是"帮助主角"） | ≥1个独立于主角的个人目标 |
| **背景故事** | 角色的过去经历/出身 | ≥200字的背景交代 |
| **性格矛盾** | 角色性格中的内在冲突点 | ≥1个性格矛盾（如外表粗犷内心细腻） |
| **成长弧线** | 角色在卷内的变化/成长 | ≥1个可观察的变化 |
| **专属高光** | 角色独立解决某个问题/展现独特价值 | 每30章≥1次独立高光时刻 |
| **关系网** | 与主角之外的其他角色的互动 | ≥2条非主角关系线 |

#### 检测规则

| 检测项 | 规则 | 级别 |
|--------|------|:--:|
| **工具人化** | 角色出场≥5章，但所有行为均为"帮助主角/被主角救/提供信息"，无独立动机 | P0 |
| **背景空白** | 主要角色（出场≥10章）的背景故事交代<200字 | P0 |
| **性格单薄** | 角色仅有1个性格标签（如"憨厚""高冷"），无性格矛盾或层次 | P1 |
| **无成长弧线** | 角色出场≥20章，但性格/能力/地位无任何变化 | P1 |
| **高光缺失** | 主要角色连续30章无独立解决问题/展现独特价值的场景 | P1 |
| **关系单一** | 角色仅与主角有互动，与其他角色无任何关系线 | P2 |
| **对话同质化** | 不同角色的对话风格/口头禅/语气无区分度 | P1 |
| **女性角色花瓶化** | 女性角色仅作为感情线工具，无独立事业/目标/能力 | P1 |

### 十三、爽点质量诊断

> 爽点不是"发生了好事"，而是"读者期待的好事以超出预期的方式发生"。入门测试这种大事如果写得平淡，等于浪费了一个天然爽点。

#### 爽点质量模型

| 质量要素 | 说明 | 权重 |
|---------|------|:--:|
| **铺垫充分度** | 爽点发生前是否有足够的压抑/期待/悬念积累 | 25% |
| **反转/超出预期** | 结果是否超出读者预期（不是"赢了"而是"以震撼全场的方式赢了"） | 25% |
| **受众反应描写** | 是否有旁观者的震惊/质疑/打脸/崇拜等反应描写 | 20% |
| **代价/付出感** | 胜利是否有代价，让读者觉得"赢得不容易" | 15% |
| **后续影响** | 爽点是否有涟漪效应（引起势力关注/改变地位/引发新冲突） | 15% |

#### 检测规则

| 检测项 | 规则 | 级别 |
|--------|------|:--:|
| **重大事件平淡化** | 入门测试/宗门大比/秘境探险等天然爽点事件，爽点质量评分<5分 | P0 |
| **铺垫不足** | 爽点事件前3章内无任何压抑/期待/悬念积累（"突然就赢了"） | P0 |
| **缺乏受众反应** | 主角完成高光行为后，无任何旁观者反应描写（震惊/质疑/崇拜） | P1 |
| **胜利过于轻松** | 主角在重大事件中无任何代价/受伤/付出即获胜 | P1 |
| **爽点无涟漪** | 爽点事件后，主角地位/人际关系/剧情走向无任何变化 | P1 |
| **爽点类型单一** | 连续10章使用同一类型爽点（如连续打脸10次） | P2 |
| **高光时刻缺失** | 主角连续10章无任何让读者"爽到"的高光时刻 | P0 |
| **对比不足** | 主角的优秀没有通过对比来凸显（无同级对比/无前辈评价/无历史参照） | P1 |

#### 逐事件爽点质量检查

| 事件 | 章节 | 事件类型 | 铺垫 | 反转 | 受众反应 | 代价 | 后续影响 | 质量评分 |
|------|:---:|------|:---:|:---:|:---:|:---:|:---:|:---:|
| 入门测试 | 第X章 | 实力展示 | <<1-10>> | <<1-10>> | <<1-10>> | <<1-10>> | <<1-10>> | <<X/10>> |
| 宗门大比 | 第Y章 | 竞技/打脸 | <<1-10>> | <<1-10>> | <<1-10>> | <<1-10>> | <<1-10>> | <<X/10>> |


### 十四、前期铺垫充分性诊断

> 第一卷是读者决定是否追读的关键。前期必须有足够的信息密度和情节厚度，让读者感受到"这个世界很大，这个故事很长"。100章的第一卷可以丰富到300章，前提是每一章都有足够的内容支撑。

#### 章节充实度模型

| 要素 | 说明 | 最低标准 |
|------|------|---------|
| **情节推进** | 每章是否有实质性的剧情进展 | 每章≥1个情节推进点 |
| **信息增量** | 每章是否提供了新的信息（设定/人物/关系/悬念） | 每章≥2个新信息点 |
| **情感波动** | 每章是否有情感变化（紧张/感动/愤怒/期待） | 每章≥1次情感变化 |
| **人物互动** | 每章是否有≥2个角色的有意义互动 | 每章≥1场对话/互动场景 |
| **环境描写** | 每章是否有场景/氛围的描写 | 每章≥1处环境描写 |

#### 检测规则

| 检测项 | 规则 | 级别 |
|--------|------|:--:|
| **章节空洞** | 单章情节推进点=0（纯水章/纯过渡章） | P0 |
| **信息增量不足** | 连续3章每章新信息点<2个 | P1 |
| **情感平直** | 连续3章无情感波动（始终同一情绪基调） | P1 |
| **对话占比过高** | 单章对话占比>70%（"对话流水账"） | P1 |
| **场景单一** | 连续5章在同一场景（同一房间/同一地点） | P1 |
| **设定堆砌未展开** | 大纲中已设定的势力/人物/规则，在正文中提及但超过10章未展开描写 | P1 |
| **伏笔密度过低** | 前30章埋设伏笔<5个（缺乏长期钩子） | P0 |
| **势力/人物引入后闲置** | 已引入的势力/人物，超过15章无任何出场或提及 | P1 |
| **章节篇幅不足** | 单章<2000字（网文标准章节应≥3000字） | P2 |
| **过渡章过多** | 连续5章均为过渡性质（无高潮/无转折/无新信息） | P0 |
