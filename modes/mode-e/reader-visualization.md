## 读者评论模拟

> 触发词：`模拟评论`、`读者视角`。跳出作者身份，模拟不同类型读者的真实反应，预测作品的市场接受度。

### 模拟触发
输入 `模拟评论` 对指定章节进行读者评论模拟。

### 三类读者画像

| 读者类型 | 关注点 | 评分权重 | 典型反应 |
|---------|--------|---------|---------|
| **爽文读者** | 爽点密度、打脸力度、升级速度 | 40% | "这章爽不爽？打脸够不够狠？" |
| **逻辑党** | 设定合理性、剧情逻辑、伏笔回收 | 35% | "这个设定合理吗？有没有BUG？" |
| **角色粉** | 人设一致性、角色成长、情感共鸣 | 25% | "XX角色的行为符合人设吗？" |

```markdown
## 读者评论模拟报告

### 章节信息
- 章节：第23章 宗门大比
- 字数：4200字
- 核心事件：林凡在宗门大比中以筑基期修为击败金丹期对手

### 爽文读者视角 ⭐⭐⭐⭐⭐ (5/5)
**爽点体验**：
- 越级挑战成功，打脸所有看不起他的人，爽！
- 最后那句"筑基期，够了"太装逼了，我喜欢

**槽点**：
- 战斗过程可以再长一点，感觉还没看过瘾就结束了

**一句话评论**：
"这章太爽了！越级打脸永远的神！就是战斗能再详细点就好了"

**弃书风险**：低

---

### 逻辑党视角 ⭐⭐⭐ (3/5)
**逻辑检查**：
- 越级战斗的合理性解释得不错（金手指+战术+对手轻敌）
- 但金丹期和筑基期的差距应该更大，赢得有点太轻松

**BUG报告**：
- 第15章说过金丹期可以灵力外放形成护盾，这章对手完全没用
- 裁判长老的反应太慢了，不符合元婴期高手的设定

**一句话评论**：
"爽是爽，但金丹期被筑基期压着打，战力体系有点崩"

**弃书风险**：中

---

### 角色粉视角 ⭐⭐⭐⭐⭐ (5/5)
**角色表现**：
- 林凡的冷静和战术思维一如既往，人设稳定
- 苏婉儿在台下紧张握拳的细节很动人
- 反派师兄的傲慢和被打脸后的表情变化很到位

**OOC风险**：
- 无

**一句话评论**：
"林凡太帅了！苏婉儿担心他的那段我反复看了三遍"

**弃书风险**：低

---

### 综合读者评分
| 读者类型 | 评分 | 弃书风险 | 核心意见 |
|---------|------|---------|---------|
| 爽文读者 | ⭐⭐⭐⭐⭐ | 低 | 爽点到位，战斗可更详细 |
| 逻辑党 | ⭐⭐⭐ | 中 | 战力体系需加强一致性 |
| 角色粉 | ⭐⭐⭐⭐⭐ | 低 | 人设稳定，情感线动人 |
| **加权综合** | **⭐⭐⭐⭐ (4.4/5)** | **低** | — |

### 优先修复建议
1. 补充金丹期对手使用灵力护盾的描写（哪怕被林凡破解）
2. 增加裁判长老的反应描写，体现元婴期高手的洞察力
3. 战斗过程可增加1-2个回合的细节描写
```

## 多模态创作辅助可视化

> 触发词：`关系图`、`世界地图`、`时间线`、`爽点热力图`、`情绪曲线叠加`。将抽象数据转化为可视化图表，直观展示创作全貌。

### 人物关系可视化

输入 `关系图` 生成ASCII艺术风格的角色关系网络图：

```

                     │   │盟主 │◀───│长老 │        │
                     │   └──┬──┘    └─────┘        │
                     └──────┼──────────────────────┘

              ┌─────────────┼─────────────┐
              │ 师徒         │ 敌对         │ 利用

    ┌─────────────┐  ┌─────────────┐  ┌─────────────┐
    │   林凡      │  │   萧天      │  │   苏婉儿    │
    │  （主角）    │  │  （反派）    │  │  （女主）    │
    │  金丹初期    │  │  金丹后期    │  │  元婴中期    │

```

**关系图例**：
| 连线 | 含义 |
|------|------|
| `───` | 普通关系 |
| `◀──` | 单向关系（箭头指向被作用方） |
| `◀──▶` | 双向关系 |
| `╌╌╌` | 隐藏/秘密关系 |
| `─X─` | 已断裂关系 |

情绪曲线分析：

### 可视化触发词汇总

| 触发词 | 输出 | 适用场景 |
|--------|------|---------|
| `关系图` | 角色关系网络图 | 群像调度、关系梳理 |
| `世界地图` | ASCII风格世界地图 | 世界观展示、空间规划 |
| `时间线` | 多线叙事时间轴 | 多线叙事协调、伏笔追踪 |
| `爽点热力图` | 爽点分布热力图 | 节奏诊断、爽点规划 |
| `情绪曲线` | 情绪+期待值叠加曲线 | 单章节奏分析、情绪管理 |
