# 模式E-灵感碰撞分析模块

> **版本**：v21.0
> **触发词**：`灵感碰撞`、`编辑评估`、`殿堂级评估`、`专业评审`、`深度分析`、`碰撞分析`
> **所属**：模式E统一诊断中心扩展模块

---

## 功能概述

灵感碰撞是一个创新性的内容评估功能，在大纲、细纲或正文创作完成后自动触发。该功能模拟番茄小说、起点中文网等平台资深编辑的视角和评判标准，从多个维度对作品进行全面分析评估。

### 核心价值

| 维度 | 描述 |
|------|------|
| **文学性评估** | 语言风格、叙事技巧、主题深度 |
| **故事架构** | 结构完整性、节奏把控、情节逻辑 |
| **人物塑造** | 角色立体度、成长弧线、动机合理性 |
| **情节逻辑** | 因果链条、伏笔回收、冲突设计 |
| **市场潜力** | 题材热度、读者定位、商业化前景 |

## 触发机制

### 自动触发条件

| 场景 | 触发时机 | 提示文案 |
|------|---------|---------|
| 大纲完成 | 模式B生成完成后 | "是否进行灵感碰撞分析？" |
| 细纲完成 | 模式B-2生成完成后 | "是否进行灵感碰撞分析？" |
| 正文完成 | 模式C单章/多章完成后 | "是否进行灵感碰撞分析？" |

### 手动触发

- 输入 `灵感碰撞` 直接启动分析
- 输入 `编辑评估` 进入评估流程
- 输入 `殿堂级评估` 进行高标准评审

## 评估维度详解

### 一、文学性评估（20%权重）

| 指标 | 评分标准 | 评估要点 |
|------|---------|---------|
| 语言风格 | 1-10分 | 文字流畅度、用词精准度、句式多样性 |
| 叙事技巧 | 1-10分 | 视角运用、节奏把控、场景转换 |
| 主题深度 | 1-10分 | 主题挖掘、思想内涵、情感共鸣 |
| 风格独特性 | 1-10分 | 辨识度、创新点、个人特色 |

### 二、故事架构（20%权重）

| 指标 | 评分标准 | 评估要点 |
|------|---------|---------|
| 结构完整性 | 1-10分 | 开头、发展、高潮、结尾完整性 |
| 节奏把控 | 1-10分 | 张弛有度、快慢适宜、信息密度 |
| 悬念设置 | 1-10分 | 钩子设计、悬念层次、揭秘节奏 |
| 情节复杂度 | 1-10分 | 支线处理、多线叙事、逻辑层次 |

### 三、人物塑造（20%权重）

| 指标 | 评分标准 | 评估要点 |
|------|---------|---------|
| 主角立体度 | 1-10分 | 性格复杂度、内心世界、成长弧线 |
| 配角丰富度 | 1-10分 | 配角功能、人物关系、群像刻画 |
| 动机合理性 | 1-10分 | 行为逻辑、目标明确、情感驱动 |
| 人物辨识度 | 1-10分 | 独特标签、语音特色、行为模式 |

### 四、情节逻辑（20%权重）

| 指标 | 评分标准 | 评估要点 |
|------|---------|---------|
| 因果链条 | 1-10分 | 前因后果、逻辑闭环、无漏洞 |
| 伏笔管理 | 1-10分 | 埋设自然、回收及时、伏笔密度 |
| 冲突设计 | 1-10分 | 冲突层次、张力强度、解决方式 |
| 合理性验证 | 1-10分 | 符合世界观、人物行为一致、无机械降神 |

### 五、市场潜力（20%权重）

| 指标 | 评分标准 | 评估要点 |
|------|---------|---------|
| 题材热度 | 1-10分 | 当前市场趋势、读者需求匹配度 |
| 目标受众 | 1-10分 | 读者定位清晰、受众画像明确 |
| 商业化潜力 | 1-10分 | IP开发价值、衍生可能性、付费转化 |
| 差异化优势 | 1-10分 | 创新点、竞争优势、独特卖点 |

## 评估等级体系

### 综合评分与等级

| 等级 | 分数范围 | 评价 | 殿堂级水准 |
|------|---------|------|-----------|
| S+ | 95-100 | 传世之作 | ✅ 殿堂级作家水准 |
| S | 90-94 | 精品佳作 | ✅ 准殿堂级 |
| A | 80-89 | 优秀作品 | ⚠️ 接近殿堂级 |
| B | 70-79 | 良好作品 | 📈 有提升空间 |
| C | 60-69 | 及格作品 | ⚠️ 需要改进 |
| D | <60 | 待优化 | 🔴 需大幅修改 |

### 编辑评语模板

**S级评语**：
> "这是一部具有殿堂级水准的作品！叙事精妙、人物鲜活、情节环环相扣。语言风格独特且成熟，主题挖掘深刻，具备成为现象级作品的潜力。强烈建议尽快完本并推向市场。"

**A级评语**：
> "作品整体质量优秀，已接近殿堂级水准。故事架构完整，人物塑造立体，情节逻辑严密。建议在[具体维度]上进一步打磨，有望冲击更高等级。"

**B级评语**：
> "作品具备良好基础，但距离殿堂级水准仍有提升空间。建议加强[具体维度]，优化[具体问题]，相信经过打磨后会有显著进步。"

## 执行流程

```
用户完成创作 → 自动触发提示 → 用户确认 → 加载多维度数据 → 执行分析
    ↓
生成评估报告 → 展示编辑点评 → 提供改进建议 → 用户选择下一步
```

### 具体步骤

1. **数据收集**：读取大纲、细纲、正文文件
2. **维度分析**：按5大维度进行深度评估
3. **等级评定**：计算综合评分，确定等级
4. **评语生成**：根据等级生成编辑评语
5. **建议生成**：针对薄弱环节提出改进建议
6. **报告输出**：格式化输出完整报告
